Η ερώτηση:

2. Για τα ακόλουθα τμήματα ενός προγράμματος Fortran θα παρουσιαστεί είτε κάποιο σφάλμα ή θα αποθηκευτούν κάποιες τιμές. Υποθέτοντας default Fortran types για κάθε μεταβλητή η οποία δεν έχει δηλωθεί μαντέψτε την απάντηση του προγράμματος:  

       REAL C, D, F
       C = 4 
       D = 2
       F = -C**2 * 1.0E+03 / 2.*D

4000.     -4000.     16000.     -16000.     Σφάλμα. Δεν θα σωθεί τίποτα

Aυτό το πρόβλημα έχει να κάνει με τους κανόνες ισχυρότητας-σειράς όπως και στην Άλγεβρα.

Αν αντικαταστήσουμε τις τιμές των μεταβλητών θα πάρουμε την ακόλουθη έκφραση:


      F = -4**2 * 1.0E+03 / 2. * 2
Η σειρά με την οποία θα εκτελεστούν οι πράξεις είναι: πρώτα το εκθετικό, κατόπιν ο πολλαπλασιασμός και διαίρεση που επειδή έχουν την ίδια βαρύτητα ακολουθείτε η εκτέλεση της πράξης που συναντάται πρώτη κινούμενοι από αριστερά προς τα δεξιά στην αλγεβρική έκφραση. Τελευταίες εκτελούνται οι προσθέσεις και αφαιρέσεις. Επομένως πρώτα θα εκτελεστεί το εκθετικό:

      F = -16 * 1.0E+03 / 2. * 2
Κατόπιν θα κάνουμε τους πολλαπλασιασμούς και διαιρέσεις από αριστερά στα δεξιά:
      F = -16000. / 2. * 2

      F = -8000. * 2 

      F = -16000.
