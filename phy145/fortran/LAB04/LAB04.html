  1. Χρησιμοποιώντας το file της 1ης άσκησης καθώς και το αλγόριθμο ταξινόμησης που είδαμε στις διαλέξεις γράψτε τα στοιχεία του πίνακα σε ένα file αλλά ταξινομημένα σε αύξουσα σειρά. Χρησιμοποιώντας κατάλληλο format γράψτε τα στοιχεία αυτά σε μορφή ενός δυσδιάστατου πίνακα με 5 στήλες (και επομένως 4 γραμμές).
    Υπόδειξη: Η εντολή Fortran write(6,*)Α έχει σαν αποτέλεσμα να γράψει ολόκληρο το πίνακα Α σε μια γραμμή. Επομένως θα πρέπει να δώσετε ένα format το οποίο θα τυπώνει αρχικά 5 θέσεις του πίνακα Α σε μια σειρά και κατόπιν θα πρέπει να αλλάζει σειρά για να τυπώσει άλλες 5 θέσεις κοκ. Για να το κάνουμε αυτό η εντολή format θα έχει τη μορφή FORMAT(5(1x,I3),/) που σημαίνει ότι τύπωσε 5 νούμερα με τον ίδιο τρόπο (1κενό, ακέραιος 3 ψηφίων) και αμέσως μετά άλλαξε γραμμή εξαιτίας του "/". Σημειωτέον ότι αν προσπαθούσαμε να κάνουμε κάτι τέτοιο μέσα σε do loop και γράφαμε
    do i=1, 20
        write(6,*)a(i)
    enddo
    τότε θα παίρναμε όλες τις τιμές σε μια στήλη αφού για κάθε εντολή write ή print το πρόγραμμα αλλάζει σειρά.
    Τέλος θα μπορούσαμε να τυπώσουμε τα νούμερα του πίνακα σα να ήταν δυσδιάστατος με τον ακόλουθο τρόπο:
    do i=1,20, 5
         write(6,    10)(a(j), j=i, i+4)
    enddo
    Με το τρόπο αυτό έχουμε ένα εξωτερικό loop ως προς τις θέσεις του πίνακα Α το οποίο αυξάνει κατά 5 (δηλαδή 1, 6, 11, 16 - και επομένως δίνει τα στοιχεία Α(1),Α(6),Α(11) και Α(16) του πίνακα). Στη 2η σειρά με την εντολή write έχουμε ένα έμμεσο loop. Βλέπετε ότι ο δείκτης j παίρνει τιμές από i (επομένως 1,6,11,16) και αυξάνει μέχρι τη τιμή Ι+4 (δηλαδή 5, 10, 15, 20) καλύπτοντας έτσι τις ενδιάμεσες θέσεις του πίνακα. H εντολή FORMAT στη περίπτωση αυτή θα έχει τη μορφή FORMAT(5(1x,I3)). Παρατηρήστε ότι εδώ δεν χρειάζεται να δώσουμε το σύμβολο "/" για αλλαγή γραμμής μια η εντολή write βρίσκεται μέσα σε ένα loop. Επομένως θα εκτελεστεί 4 φορές και κάθε φορά που εκτελείται το πρόγραμμα θα αλλάζει αυτόματα σειρά.
