Οδηγίες για την εγκατάσταση του πακέτου cygwin

 

Ακολουθήστε τις παρακάτω οδηγίες μόνο αν έχετε υπολογιστή (laptop ή Desktop PC) που τρέχουν κάποιο version των WINDOWS

  • Επισκεφθείτε την σελίδα http://www.cygwin.com/

  • Κάντε click στο σημείο που λέει Install or update now! (using setup.exe)

  • Στο παράθυρο που θα ανοίξει επιλέξτε Save

  • Θα σας ρωτήσει που θα θέλατε να σώσετε το setup.exe. Επιλέξτε ένα directory που θέλετε (π.χ. C:\Cygwininstall, [πρέπει να έχετε δημιουργήσει πρώτα το directory αυτό] και πατήστε Save. To file θα αποθηκευτεί και ένα άλλο παράθυρο θα ανοίξει ζητώντας αν θέλετε να τρέξετε το setup.exe. Επιλέξτε run. Θα εμφανιστεί το παρακάτω παράθυρο

  • Επιλέξτε Νext>. Στην επόμενη ερώτηση αφήστε το Default: (Install from internet).

  • To ίδιο και για την επόμενη ερώτηση. Αφήστε τις επιλογές όπως είναι (Root directory: C:\cygwin , Install for all users και Default Text File Unix).

  • Στην ερώτηση για επιλογή Local Package Directory πληκτρολογήστε και πάλι C:\Cygwininstall ή κάντε browse σ’ αυτό το directory.

  • Αφήστε το default για το είδος της σύνδεσης (direct connection).

  • Θα σας ρωτήσει από που θέλετε να κατεβάσετε τα αρχεία. Επιλέξτε μια σελίδα (δοκιμάστε http://www.mirrorservice.org ή http://mirrors.kernel.org) και πατήστε Νext>

  • H επόμενη ερώτηση είναι η πιο δύσκολη μια και πρέπει να επιλέξετε τα πακέτα που θα χρειαστεί να κατεβάσετε. Αν δεν επιλέξετε τίποτα και πατήσετε Νext θα κατεβάσει μόνο αυτά που βρίσκονται στην κατηγορία Base. Ωστόσο οι compilers, το gnuplot και οι editors δεν περιέχονται στο βασικό αυτό πακέτο. Για το λόγο αυτό θα χρειαστεί να κάνετε τις ακόλουθες επιλογές:

  • Πηγαίνετε το ποντίκι στο τελευταίο πακέτο (Χ11) και πιέστε μια φορά το σύμβολο δίπλα από τη λέξη Default

To κλικ αυτό θα αλλάξει τη λέξη από Default σε Install.  Αυτό είναι το μόνο πακέτο που πρέπει να κάνετε install ολόκληρο. Για τα υπόλοιπα πακέτα τα οποία σας αναφέρω παρακάτω θα πρέπει να κάνετε install μόνο μέρος των προγραμμάτων που περιέχουν.  Για να γίνει αυτό θα πρέπει να πάτε πάνω από το σύμβολο στα αριστερά των πακέτων (+) και κάντε click στο σύμβολο αυτό. Θα γίνει expand το πακέτο όπως παρακάτω: (έχω δοκιμάσει το πακέτο Games για παράδειγμα)

  • Όπως βλέπετε υπάρχουν διάφορα προγράμματα. Πηγαίνετε για παράδειγμα στη γραμμή που γράφει sudoku και με το mouse κάντε μια φορά click στο κυκλικό σύμβολο με τα δυο τόξα. Η λέξη από Skip θα αλλάξει σε κάποιο νούμερο που αντιστοιχεί στη τελευταία version του προγράμματος όπως παρακάτω που εμφανίζεται 1.0.1-1 ενώ εμφανίζεται και ένα τετραγωνάκι με ένα x που δηλώνει ότι θα κατεβάσετε το binary του προγράμματος.  

  • Τη παραπάνω διαδικασία θα πρέπει να ακολουθήσετε για τα παρακάτω πακέτα και προγράμματα που σας δίνω στον πίνακα που ακολουθεί. Σε μερικές περιπτώσεις κάποια από τα προγράμματα που αναφέρονται επιλέγονται αυτόματα μόλις επιλέξετε κάποιο προηγούμενο πρόγραμμα οπότε αν τα δείτε να είναι επιλεγμένα απλά προχωρήστε στο επόμενο. Αν κατά λάθος κάνατε 2 φορές click απλά συνεχίστε να κάνετε click μέχρι να πάρετε τη λέξη skip. Το επόμενο click θα σας δώσει το τελευταίο version που θέλατε από την αρχή.

  • Αφού τελειώσετε με την επιλογή των παραπάνω προγραμμάτων πιέστε Νext στο κάτω δεξιά γωνία του παράθυρου. Αρχίζει να κατεβάζει τα διάφορα λογισμικά και κατόπιν αρχίζει η εγκατάστασή τους. Ανάλογα με την ταχύτητα της σύνδεσης μπορεί να πάρει γύρω στα 45min για να κατεβάσει όλα τα προγράμματα οπότε και μετά ξεκινά η εγκατάστασή τους που παίρνει γύρω στα 20 λεπτά και ακολουθεί η επιβεβαίωσή τους (άλλα 10-15 λεπτά). Στο τέλος της διαδικασίας αυτής σας ρωτά αν θέλετε να δημιουργηθεί ένα shortcut στο desktop και ένα shortcut στο startup. Επιλέξτε μόνο το shortcut στο desktop.

  • Μετά την εγκατάσταση θα έχετε ένα εικονίδιο με το όνομα Cygwin στο desktop. Κάντε δεξί click με το ποντίκι στο εικονίδιο αυτό και επιλέξτε τις ιδιότητες (properties).  Θα πάρετε ένα καινούργιο παράθυρο και στο τετράγωνο που αναφέρει "Target" αντικαταστήστε αυτό που έχει με το C:\cygwin\bin\strartxwin.bat. Πατήστε ok.

  • Κάντε τώρα διπλό αριστερό click στο εικονίδιο του cygwin που έχετε στο Desktop. Αυτό θα σας δώσει ένα παράθυρο όμοιο με αυτό που έχετε στους υπολογιστές του εργαστηρίου (ok περίπου όμοιο).  Με το που ανοίγει το παράθυρο είναι σα να κάνετε login σε ένα Linux/Unix σύστημα.  Την πρώτη φορά που θα τρέξει θα δημιουργήσει ένα directory με όνομα, το όνομα του χρήστη που έχετε στα WINDOWS. Ο directory αυτός βρίσκεται στο C:\cygwin\home\<username>.

  • Κάτω από το directory αυτό μπορείτε να δημιουργήσετε τους δικούς σας directories και να τρέξετε οτιδήποτε χρησιμοποιώντας Unix εντολές.  Κάθε φορά που κάνετε διπλό click στο σύμβολο του cygwin θα σας ανοίγει ένα νέο terminal παράθυρο. Αν θέλετε να έχετε διαφορετικό χρώμα στο παράθυρο του terminal μπορείτε να αντιγράψετε το αρχείο .Xdefaults στο home directory που δημιουργήθηκε (αυτό είναι στο C:\cygwin\home\<username>). Προσέξτε την (".") τελεία στην αρχή του file αυτού. Θα πρέπει να το αποθηκεύσετε με αυτό ακριβώς το όνομα. [Πολλές φορές τα Windows δεν "αρέσουν" ονόματα που ξεκινούν με τελεία οπότε πιθανόν να το αποθηκεύσει με το όνομα [1].Xdefaults Απλά μετονομάστε το σε .Xdefaults χρησιμοποιώντας την εντολή Unix (από το παράθυρο terminal) mv [1].Xdefaults .Xdefaults.

  • Πληκτρολογώντας emacs θα ανοίξει ένα νέο παράθυρο με τον emacs editor και μπορείτε να γράψετε κάποιο πρόγραμμα. Για να το κάνετε compile πληκτρολογήστε g77 -o <name>.x <name>.f , όπου <name> το όνομα του αρχείου που δημιουργήσατε. Για να το τρέξετε πληκτρολογήστε το όνομα του executable, δηλαδή ./<name>.x 

  • Πολλοί αρέσουν τα διαφορετικά χρώματα που σας δίνει ο emacs όταν γράφετε ένα πρόγραμμα. Για να μπορέσετε να ενεργοποιήσετε αυτό το χαρακτηριστικό του emacs χρειάζεται να κατεβάσετε το .emacs (προσοχή και πάλι την τελεία "." στην αρχή του ονόματος) στο home directory σας (C:\cygwin\home\<username>). Το file αυτό μπορείτε να το βρείτε εδώ (κάντε δεξί click και μετά save target as. Θα πρέπει να το σώσετε στο home directory σας. Αρκετές φορές τα windows θέλουν να βάλουν [1].emacs στην αρχή. Αν συμβεί αυτό τότε από το terminal window αλλάξτε το όνομα του με την εντολή mv [1].emacs  .emacs)

Αν θελήσετε αργότερα να κάνετε unistall ότι βάλατε απλά πάρτε όλο το directory cygwin και μεταφέρετε το στο Trash.
