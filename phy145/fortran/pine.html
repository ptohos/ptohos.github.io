Ακολουθήστε τις παρακάτω οδηγίες ώστε να μπορείτε να στέλνετε e-mails χρησιμοποιώντας τα accounts σας στο φυσικό ή στους υπολογιστές του εργαστηρίου της πληροφορικής. O mailer που χρησιμοποιείται λέγεται pine. Περισσότερες πληροφορίες μπορείτε να βρείτε σε αυτή τη διεύθυνση. Τα παρακάτω είναι οι βασικές εντολές ώστε να ξεκινήσετε και να μπορείτε να στέλνετε κάποια e-mails χωρίς να καταφεύγετε στα e-mails στους διάφορους Internet providers (π.χ. hotmail, yahoo, cytanet κλπ)

Ξεκινώντας Pine:

  • Για να ξεκινήσετε: Ανοίξτε ένα παράθυρο terminal (όπως αυτό στο οποίο κάνετε compile τα προγράμματά σας) και πληκτρολογήστε την εντολή pine. Την πρώτη φορά που θα ξεκινήσετε pine θα σας δώσει ένα μήνυμα που σας καλωσορίζει στο περιβάλλον του pine. Μπορείτε να βγείτε από το μήνυμα αυτό πληκτρολογώντας Ε. Αυτό θα σας φέρει στο κύριο menu του pine που μοιάζει όπως φαίνεται παρακάτω:

H πάνω γραμμή σας λέει ότι βρίσκεται στο κύριο menu και πόσα e-mails υπάρχουν στο φάκελο των εισερχομένων μηνυμάτων. Οι υπόλοιπες γραμμές σας δίνουν τις βασικές επιλογές του pine. Στο κάτω μέρος του menu υπάρχουν διάφορες άλλες εντολές που μπορείτε να χρησιμοποιήσετε πληκτρολογώντας το σκιασμένο γράμμα ή σύμβολο κάθε φορά.

Για να βγείτε από το pine: πληκτρολογήστε το γράμμα Q (που σημαίνει quit)

 

Το Kύριο Menu:

  • Για να επιλέξετε κάποια εντολή από τo κύριο menu, κινηθείτε με τα βέλη (πάνω ή κάτω) μέχρι να σκιαστεί η γραμμή της εντολής που θέλετε να επιλέξετε και κατόπιν πιέστε το πλήκτρο return.  Θα μπορούσατε να κάνετε το ίδιο πληκτρολογώντας μόνο το γράμμα της αντίστοιχης εντολής. Για παράδειγμα για να γράψετε ένα e-mail (compose and post a message) επιλέξτε τα βέλη την εντολή C και πατήστε return ή απλά πληκτρολογήστε C.

  • Για να γράψετε κάποιο μήνυμα πατήστε C. Αυτό θα σας φέρει σε ένα καινούργιο window που αντιστοιχεί στην οθόνη δημιουργίας μηνύματος (όπως φαίνεται παρακάτω):

(Το:) σας ζητά να πληκτρολογήσετε την ηλεκτρονική διεύθυνση του ατόμου στο οποίο θέλετε να στείλετε το μήνυμα σας. (Δοκιμάστε κάποιο φίλο σας ή στο account σας σε hotmail ή yahoo αν έχετε).

(CC:) σας ρωτά αν θέλετε να στείλετε ακριβές αντίγραφο (Carbon Copy) του μηνύματος σε κάποιο χρήστη. Δεν είναι απαραίτητο να στείλετε copy σε κάποιο άλλον χρήστη. Σημειωτέον ότι όταν στέλνετε ένα μήνυμα αυτό στέλνετε αυτόματα στον εαυτό σας. Αν θέλετε να βάλετε περισσότερους από ένα χρήστες που λαμβάνουν το μήνυμα σας μπορείτε να γράψετε τη λίστα των ηλεκτρονικών διευθύνσεων χωρισμένες με κόμμα.

(Αttchmnt:) τυχόν αρχεία που θέλετε να προσκολλήσετε στο μήνυμα σας. Προσοχή εδώ ότι για να βρεί το αρχείο που θέλετε να προσκολλήσετε θα πρέπει να δώσετε την πλήρη θέση (directory path) στην οποία βρίσκεται το αρχείο σας. Για παράδειγμα, αν έχετε κάποιο αρχείο που ονομάζεται test.f που βρίσκεται στον υποφάκελο LAB04 του home φακέλου σας θα πρέπει να πληκτρολογήσετε: ~<user name>/LAB04/test.f  και πιέστε return (όπου <username> είναι το όνομα του account σας). Aν θέλετε να στείλετε περισσότερα από ένα αρχεία απλά βάλτε κόμμα στο τέλος του ενός αρχείου ακολουθούμενο από το full path του επόμενου αρχείου όπως και παραπάνω. Στο τέλος πιέστε return.

(Subject:)  εδώ γράφεται σύντομα το θέμα του μηνύματος σας (δεν είναι απαραίτητο) π.χ. homework 4 και πιέστε return.

Message text: Εδώ πλέον γράφεται το κύριο σώμα του μηνύματος σας. Σημειωτέον ότι θα μπορούσατε να αγνοήσετε όλες τις προηγούμενες εντολές και μετά την γραφή της ηλεκτρονικής διεύθυνσης του παραλήπτη θα μπορούσατε με το βέλος προς τα κάτω να πάτε απ' ευθείας στο κύριο μέρος. Το ίδιο μπορείτε να κάνετε και αφού γράψετε το μήνυμα σας να ανεβείτε με το βέλος προς πάνω και να συμπληρώσετε κάποια άλλα στοιχεία που αφήσατε αρχικά (πχ. CC, ή κάποιο attachment)

 
Όπως βλέπετε το κάτω τμήμα της οθόνης περιέχει μια σειρά από εντολές που θα μπορούσατε να χρησιμοποιήσετε. Κάθε εντολή ξεκινά με ένα ^ ακολουθούμενη από ένα λατινικό γράμμα και την εξήγηση της εντολής. Το σύμβολο ^ σημαίνει ότι πρέπει να πιέσετε το πλήκτρο ctrl (control) και ταυτόχρονα το αντίστοιχο πλήκτρο με το λατινικό γράμμα της εντολής Για παράδειγμα για να στείλετε το μήνυμα που γράψατε θα πρέπει να πιέσετε ctrl και Χ ταυτόχρονα. Για να ακυρώσετε το μήνυμα σας πιέστε ctrl και C και θα σας ζητήσει επιβεβαίωση της εντολής. Δεν είναι απαραίτητο να χρησιμοποιήσετε κεφαλαία ή μικρά γράμματα. Αν αλλάξετε γνώμη και δεν θέλετε να στείλετε το μήνυμα αλλά δεν θέλετε να το ακυρώσετε μπορείτε να πιέσετε ctrl O (postpone) και να το συνεχίσετε αργότερα.
 
Μπορείτε πλέον να δοκιμάσετε τις διάφορες εντολές που φαίνονται για να εξοικειωθείτε με το πρόγραμμα.
 
Πως βλέπετε, διαβάζετε, απαντάτε και μεταφέρετε τα μηνύματα σας:
 
To πρόγραμμα κρατά τα μηνύματα που σας στέλνονται σε ένα φάκελο που λέγεται inbox. Tα διάφορα μηνύματα μένουν στον φάκελο αυτό μέχρι να σβήσετε, ή να μεταφέρετε σε άλλο φάκελο.
Λίστα μηνυμάτων: Για να δείτε τη λίστα των μηνυμάτων σας στο inbox, πιέστε Ι (message index) όταν βρίσκεστε στο κύριο menu του pine. (Αν βρίσκεστε σε κάποιο άλλο φάκελο μπορείτε να πιέσετε Μ και θα γυρίσετε στο Main Menu).  Aν έχετε κάποιο μήνυμα, θα τα δείτε στη λίστα που παρουσιάζεται στην οθόνη των inbox μηνυμάτων. (δείτε το παρακάτω παράδειγμα)
 

 

Βλέπετε ότι στο αριστερό μέρος του κάθε μηνύματος υπάρχει ένα γράμμα που σας λέει την κατάσταση του μηνύματος. Για παράδειγμα στο πρώτο μήνυμα υπάρχει ένα D (που σημαίνει ότι το μήνυμα θα σβηστεί (deleted)). Στο δεύτερο μήνυμα υπάρχει ένα Α που δηλώνει ότι το μήνυμα έχει απαντηθεί προς τον αποστολέα του μηνύματος. Τέλος στο τρίτο μήνυμα υπάρχει ένα Ν το οποίο δηλώνει ότι μήνυμα αυτό είναι νέο (New) και δεν το έχετε διαβάσει ακόμα. Όταν το διαβάσετε και δεν θέλετε να κάνετε κάτι ιδιαίτερο (Reply, Forward, Delete) το μήνυμα δεν θα έχει κάποιο γράμμα σχετιζόμενο μαζί του. Όπως βλέπετε στο κάτω μέρος της οθόνης υπάρχει μια σειρά από εντολές στο ίδιο στυλ με τις εντολές που έχουμε δεί μέχρι τώρα. Πιέζοντας το κατάλληλο γράμμα ή σύμβολο επιλέγετε την αντίστοιχη εντολή. Βλέπετε ότι υπάρχει και μια επιλογή Ο (other CMDS)  η οποία σας και περισσότερες επιλογές που δεν φαίνονται στην αρχική οθόνη.  

 

Για να διαβάσετε κάποιο μήνυμα: Κινηθείτε με τα πλήκτρα των βελών στο μήνυμα που θέλετε και πιέστε return.Θα μπορούσατε να πατήσετε το πλήκτρο V (view). Για να κινηθείτε στο επόμενο ή στο προηγούμενο μήνυμα πατήστε το γράμμα P (previous) ή το γράμμα Ν (next).  Για να απαντήσετε ή να μεταφέρετε κάποιο μήνυμα σας πιέστε R (reply) ή F (Forward) και ακολουθήστε τις οδηγίες που εμφανίζονται.

Για να σώσετε κάποιο μήνυμα σας σε αρχείο (hard copy):  Πιέστε το γράμμα Ε (export). (Η εντολή βρίσκεται αν πιέσετε την επιλογή Ο (other CMDS) ενώ βλέπετε το μήνυμα). Το πρόγραμμα θα σας ρωτήσει να δώσετε το όνομα του αρχείου που θέλετε να αποθηκεύσετε το μήνυμα. Προσοχή αν το αρχείο υπάρχει θα αντικατασταθεί από το νέο αρχείο που γράφεται.
 
Για να δείτε προσκολλημένα αρχεία: Αν στο μήνυμα που διαβάζετε υπάρχει προσκολλημένο αρχείο τότε το πρόγραμμα θα σας ενημερώσει για την ύπαρξή του στην αρχή του μηνύματος που διαβάζετε. (Part/Attachments): ακολουθούμενη από μια λίστα από τα προσκολλημένα αρχεία. Κινούμενοι με τα πλήκτρα των βελών μπορείτε να επιλέξετε το συγκεκριμένο αρχείο και κατόπιν πιέστε V (view). Συνήθως τα αρχεία που μπορείτε να δείτε είναι text files (όπως τα αρχεία της Fortran), postscript και PDF files. Aν το πρόγραμμα δεν αναγνωρίζει το είδος του αρχείου θα σας δώσει ένα μήνυμα ότι δεν μπορεί να ανοίξει τον viewer για την συγκεκριμένη application. Ωστόσο μερικά από τα αρχεία μπορείτε να τα δείτε αφού σώσετε το προσκολλημένο αρχείο στο δίσκο και χρησιμοποιώντας την ανάλογη application. Για να σώσετε το προσκολλημένο αρχείο κάπου στο δίσκο (hard copy) επιλέξτε το αρχείο και πατήστε το γράμμα S (save). To πρόγραμμα θα σας ρωτήσει που θέλετε να σώσετε το συγκεκριμένο αρχείο (από μόνο του κρατά το όνομα του αρχείου όπως είναι στην προσκόλληση και το γράφει στο home φάκελό σας).
 
Για να διαγράψετε κάποιο μήνυμα:  Μπορείτε ενώ διαβάζετε το μήνυμα να πιέσετε το γράμμα D (delete) ή όταν βλέπετε τη λίστα με τα μηνύματα σας (δείτε την τρίτη εικόνα παραπάνω) να επιλέξετε μήνυμα που θέλετε να διαγράψετε χωρίς να το διαβάσετε, πιέζοντας το γράμμα D. Θα δείτε ότι δίπλα (αριστερά) του μηνύματος θα εμφανιστεί το γράμμα D που όπως είπαμε σημαίνει ότι το μήνυμα έχει σημειωθεί για διαγραφή. Αν αλλάξετε γνώμη μπορείτε να πιέσετε το γράμμα U (undelete). Σημειωτέον ότι τα μηνύματα παρ' όλο που σημειώνονται προς διαγραφή δεν διαγράφονται μέχρι να πείτε του προγράμματος να τα διαγράψει. Αυτό γίνεται με την εντολή expunge. Για να εκτελέσετε την εντολή αυτή θα πρέπει να βρίσκεστε στην οθόνη της λίστας των μηνυμάτων (εικόνα 3 παραπάνω) και να πιέσετε το γράμμα Χ (expunge). H εντολή βρίσκεται στην τρίτη σειρά εντολών που μπορείτε να πάρετε αν πιέσετε την Ο (other CMDS)) εντολή. To πρόγραμμα θα σας ρωτήσει αν θέλετε όντως να κάνετε expunge τα συγκεκριμένα μηνύματα. Η ίδια διαδικασία (δηλαδή η expunge process) ενεργοποιείται αυτόματα την στιγμή που βγαίνετε από το πρόγραμμα. Δηλαδή όταν λέτε Q (quit). To πρόγραμμα θα σας ρωτήσει αν θέλετε να διαγράψετε οριστικά (expunge) τα μηνύματα που έχετε σημειώσει για διαγραφή. Αν απαντήσετε όχι τότε τα μηνύματα θα παραμείνουν ως έχουν αλλά πάντοτε σημειωμένα προς διαγραφή.
 
Όπως όλα τα προγράμματα, έτσι και το pine χρειάζεται κάποια εξάσκηση για να μπορείτε να αισθάνεστε άνετα και να ξέρετε πως δουλεύουν οι διάφορες εντολές.  Τα παραπάνω ήταν μια απλή εισαγωγή στο πρόγραμμα με τις βασικότερες εντολές. Επαφίεται σε σας να προσπαθήσετε να μάθετε περισσότερα.
