Γραφήματα δεδομένων με error bars

Αν χρειάζεται να προσθέσετε error bars στο γράφημά σας, χρησιμοποιείστε την εντολή:

           plot  "my-data-file" with  <error-display>

Οι επιλογές για <error-display> είναι errorbars, xerrorbars και yerrorbars.

Τα σφάλματα λαμβάνονται από το ίδιο το file των δεδομένων "my-data-file". Για τις ανάγκες μας, έστω ότι η πρώτη και δεύτερη στήλη περιέχει τις τιμές των x και y. Στην περίπτωση αυτή το πρόγραμμα υποθέτει ότι η πληροφορία σχετικά με το σφάλμα μπορεί να βρίσκεται με τις μορφές:

              x       y      dy                                               (yerrorbars)
              x       y      ylow     yhigh                               (yerrorbars)
              x       y      dx                                               (xerrorbars)
              x       y      xlow     xhigh                               (xerrorbars)
              x       y      dx         dy                                   (errorbars)
              x       y      xlow     xhigh    ylow   yhigh         (errorbars)

H αντιστοιχία μεταξύ της επιλεγμένης δυνατότητας και του αριθμού των στηλών είναι συγκεκριμένη και μη αμφιλεγόμενη για κάθε περίπτωση.

Παράδειγμα:      plot "data3.dat" every 20 with errorbars
