Γραφήματα συναρτήσεων με το gnuplot

H εντολή για την κατασκευή ενός 2-Δ γραφήματος είναι η plot. Ενεργεί σε κάποιο αρχείο δεδομένων ή κάποια συγκεκριμένη συνάρτηση και έχει πολλαπλές επιπλέον επιλογές. Μπορείτε να κάνετε το γράφημα μιας συνάρτησης πληκτρολογώντας κάτι σαν το ακόλουθο:

           plot sin(x)

Πολλές συναρτήσεις μπορούν να αναπαρασταθούν γραφικά στο ίδιο γράφημα δίνοντας ένα κόμμα "," μεταξύ των ονομάτων τους. Για παράδειγμα:

           plot sin(x), cos(x)

Όλες οι συναρτήσεις της C-βιβλιοθήκης (και της Fortran) αναγνωρίζονται αλλά μπορείτε να ορίσετε και τις δικές σας συναρτήσεις όπως για παράδειγμα

           f(x) = x*x*sin(x)
           plot f(x)

H τιμή του π στο προηγούμενο παράδειγμα είναι ορισμένη στην βιβλιοθήκη του λογισμικού. Τα όρια του διαστήματος για το x-άξονα καθορίζονται αυτόματα από το gnuplot. Αν θέλετε να τα αλλάξετε τότε μπορείτε να χρησιμοποιήσετε την ακόλουθη σύνταξη: 

          plot [-2*pi:4*pi] sin(x)

(παρατηρήστε ότι το gnuplot μπορεί να εκτελέσει τις πράξεις που συναντά σε μια εντολή). Για το ίδιο αποτέλεσμα θα μπορούσατε να εκτελέσετε την εντολή:

         set xrange [-2*pi:4*pi]

πριν την εντολή plot και να επιτύχετε το ίδιο αποτέλεσμα. Ακριβώς τα ίδια ισχύουν αν θέλετε να αλλάξετε το διάστημα που λαμβάνεται στον y-άξονα. χρησιμοποιώντας την εντολή set yrange. Αν επιθυμήτε να επαναφέρετε το διάστημα που επιλέχθηκε από το gnuplot αυτόματα τότε πληκτρολογήστε την εντολή

         set xrange restore

Για να αυξήσετε την διακριτική πιστότητα του γραφήματος χρησιμοποιήστε την εντολή

         set sample <διάστημα>

H τιμή του διαστήματος που υπάρχει στο gnuplot αυτόματα είναι 100. Αυξάνοντας τον αριθμό αυτό δίνει περισσότερο ομαλό γράφημα.

Μερικές άλλες πολύ χρήσιμες εντολές στο ίδιο πνεύμα είναι οι ακόλουθες:

          plot f(x) with lines/points

H εντολή αυτή προκαλεί το γράφημα της συνάρτησης f(x) σαν γραμμή (default) ή σημεία.

          set polar

Με την εντολή αυτή μπορούμε να δουλέψουμε σε πολικές συντεταγμένες. Η μεταβλητή στην εντολή plot μετατρέπεται αυτόματα σε γωνία theta. Χρησιμοποιώντας την εντολή set nopolar επιστρέφουμε στις καρτεσιανές συντεταγμένες.
