Γραφήματα δεδομένων

Αν αντί του ονόματος μιας συνάρτησης χρησιμοποιήσουμε το όνομα ενός file, η εντολή plot σχεδιάζει ένα γράφημα χχρησιμοποιώντας την 1η στήλη δεδομένων του file σαν τις τιμές του x-άξονα, ενώ τις τιμές της 2ης στήλης τις χρησιμοποιεί για τιμές του y-άξονα. Το όνομα του file θα πρέπει να περιέχεται σε απλά ή διπλά αποσιωπητικά. Για παράδειγμα:

           plot  "my-data-file"

θα ανοίξει ένα νέο window, θα χαράξει ένα τετράγωνο με αριθμημένους άξονες και θα κάνει ένα γράφημα στο οποίο θα δώσει ένα όνομα. Αν αντί του ονόματος ενός αρχείου "my-data-file" ήταν "-" αυτό θα ισοδυναμούσε με εισαγωγή εντολής από το πληκτρολόγιο. Αν αντί αυτών χρησιμοποιήσουμε το όνομα "<program", τότε το gnuplot τρέχει το συγκεκριμένο πρόγραμμα και δέχεται σαν input το output του προγράμματος. Αυτό αποτελεί ένα καλό τρόπο για να γράφουμε "υποπρογράμματα" για το gnuplot.

Μια περισσότερο πλήρης plot εντολή μπορεί να είναι η ακόλουθη:

           plot [x-range] [y-range]                         \
                   "my-data-file"                               \
                            every sampling-interval       \
                            using column-list                  \
                            with point-or-lines

όπου το σύμβολο ("\") συμβολίζει συνέχιση της εντολής ενώ ότι αναγράφεται με κόκκινα γράμματα αναφέρεται σε δικές μας επιλογές. Τα υπόλοιπα τμήματα της εντολής που γράψαμε έχουν την ακόλουθη σημασία:

[x-range]
      χρησιμοποιείται επιλεκτικά αν θέλουμε να ορίσουμε το διάστημα στο x ή y-άξονα.

every sampling-interval
       προσδιορίζει ποια σημεία (π.χ. γραμμές του file) να λάβει υπ' όψιν στον σχεδιασμό του γραφήματος. Κανονικά όλα τα σημεία εισέρχονται στο γράφημα αλλά αν για παράδειγμα χρησιμοποιήσουμε την τιμή 2 σαν sample-interval, τότε στο γράφημα θα εισέρχεται κάθε δεύτερη γραμμή.

using column-list
        προσδιορίζει ποιές στήλες από το file θα χρησιμοποιηθούν στην κατασκευή του γραφήματος. Αυτόματα θεωρεί τις στήλες 1:2

with point-or-lines
       προσδιορίζει το αν θα θέσει μεμονωμένα σημεία ή θα συνδέσει τα δεδομένα με γραμμές. Αυτόματα το πρόγραμμα χρησιμοποιεί μεμονωμένα σημεία

Οι εντολές μπορούν να γραφούν με συγκοπτόμενο τρόπο και πολλά γραφήματα μπορούν να γίνουν με το να δώσουμε τα ονόματα περισσότερων files, διαφορετικά διαστήματα γραμμών και στηλών ή είδος σημείων αρκεί όλα να ξεχωρίζονται με κόμμα ",". Προσοχή ωστόσο στην σειρά των διαφόρων επιλογών μια και αυτή είναι σημαντική και θα πρέπει να διατηρείται.

Παράδειγμα 1ο  

        plot [0:10] [-1:1] "data1.dat" us 1:2 w lines, \
                                   "data1.dat" ev 2 usi 1:3 with points, \
                                   "data3.dat" ev 3 using 3:2 wi points

H εντολή αυτή θα δημιουργήσει τρία γραφήματα με τους ίδιους άξονες, το πρώτο γράφημα θα είναι με γραμμές, τα επόμενα 2 γραφήματα θα είναι με σημεία, χρησιμοποιώντας δεδομένα από 2 files (data1.dat και data3.dat), δείχνοντας όλα τα δεδομένα στο πρώτο γράφημα, κάθε δεύτερο στο δεύτερο γράφημα και κάθε τρίτο στο τρίτο γράφημα.

Παράδειγμα 2ο

        plot "data2.dat" with points

Με την εντολή αυτή θα γίνει το γράφημα 5000 τυχαίων σημείων σε ένα τετράγωνο με πλευρά 10.
