Κατασκευές πολλαπλών γραφημάτων στην ίδια σελίδα

Η εντολή plot σβήνει το display πριν δημιουργήσει ένα νέο γράφημα. Για να δημιουργήσετε πολλαπλά τετράγωνα και γραφήματα στην ίδια σελίδα, χρησιμοποιείστε την εντολή 

           set multiplot 

Οποιαδήποτε μεταγενέστερη εντολή  plot δώσετε τότε το γράφημα θα σχεδιαστεί στα ήδη υπάρχοντα.

Οι ακόλουθες εντολές χρησιμοποιούνται συνήθως σε περιβάλλον multiplot.

set origin x, y
     δίνεται η αρχή (κάτω αριστερή γωνία) του επόμενου γραφήματος. Το σύστημα συντεταγμένων που χρησιμοποιείται για τα x και y είναι τέτοιο ώστε το κάτω αριστερό άκρο του display αντιστοιχεί στο (0,0) και το πάνω δεξί άκρο στο (1,1)

set size xscale yscale
      δίνεται το μέγεθος του επόμενου γραφήματος. Αν παραληφθεί το yscale τότε λαμβάνεται όσο και το xscale. H κλίμακα είναι πάντοτε σχετικά με το καθιερωμένο γράφημα που καταλαμβάνει όλο το display.

set nomultiplot
       με την εντολή αυτή σταματά το περιβάλλον των multiplot. H επόμενη εντολή plot θα σβήσει το display.
 

Παράδειγμα:

            set multiplot
            set size 0.5
            set origin 0,0
            plot 'data1.dat' every 3 using 1:3 with points
            set origin 0.4, 0.5
            set size 0.4
            plot 'data2.dat' lt 3
            set origin 0.6, 0.1
            splot 'data4.dat' every 5 lt 4
            set nomultiplot

Παρατηρήστε ότι όταν βρίσκεστε σε multiplot περιβάλλον τότε και η αρχή της γραμμής εντολής αλλάζει από "gnuplot>" σε "multiplot>" για να σας υπενθυμίζει ότι βρίσκεστε σε περιβάλλον multiplot.
