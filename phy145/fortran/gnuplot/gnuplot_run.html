Ξεκινώντας το gnuplot

Για να ξεκινήσετε το gnuplot πληκτρολογήστε σε ένα terminal window την εντολή:

       gnuplot

Αν θέλετε μπορείτε να δώσετε μετά το gnuplot τα ονόματα κάποιων files (scripts) που περιέχουν εντολές του πακέτου gnuplot. Αν θέλετε μπορείτε επίσης να κρατήσετε το window display το οποίο ανοίγει το gnuplot ακόμα και όταν το πρόγραμμα gnuplot έχει σταματήσει να τρέχει. Για να το κάνετε αυτό πληκτρολογήστε την επιλογή "-persist" μετά το gnuplot

      gnuplot -persist script1 script2 ...

Μπορείτε ακόμα να αναμίξετε εντολές που δίνονται από κάποια scripts αρχεία με εντολές που δίνονται από το πληκτρολόγιο παρεμβάλλοντας το σύμβολο "-" μεταξύ των script files. Με το τρόπο αυτό αν κάποιο γράφημα για να πραγματοποιηθεί χρειάζεται κάποια σειρά εντολών στην αρχή και στο τέλος μπορείτε να προσθέσετε τις δικές σας εντολές χρησιμοποιώντας "-". Για παράδειγμα:

     gnuplot -persist script1 - script2 ...
