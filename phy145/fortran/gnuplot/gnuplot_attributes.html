Αλλαγές στην όψη των γραφημάτων - Beautifying

Υπάρχουν πάρα πολλές εντολές για να βελτιώσουμε τα γραφήματα. Μια σύντομη λίστα χωρισμένη ανά είδος λειτουργικότητας είναι:

  • Επιλογή της μονάδας εξόδου
     

  • Διεργασίες που αφορούν τους άξονες
     

  • Επιλογή του μεγέθους του πλαισίου και του σχήματός του
     

  • Δημιουργία ετικετών στους άξονες και σε άλλα σημεία
     

  • Επιλογή του είδους της γραμμής και των σημείων
     

  • Μερικές άλλες χρήσιμες εντολές

  • clear
                  Καθαρίζει το display

  • pause wait-time "message"
              Τύπωσε κάποιο μήνυμα και περίμενε τον προκαθορισμένο αριθμό δευτερολέπτων. Αν wait-time είναι αρνητικός, περιμένει μέχρις ότου  δοθεί <Return> πριν συνεχίσει

  • replot
               Επανέλαβε την προηγούμενη εντολή plot ή splot με νέες παραμέτρους

  • reset
               Επανέφερε όλες τις επιλογές στις προκαθορισμένες (default) τιμές τους

  • show <option>
                Δείχνει τη τιμή οποιασδήποτε παραμέτρου που μπορεί να καθοριστεί με την εντολή set.
     

  • Επιλογή της μονάδας εξόδου

  • set terminal <device-type>
           θέτει την μονάδα εξόδου σε <device-type>. Οι περισσότερο συνήθεις επιλογές είναι Χ11 (το Χ-display, προκαθορισμένη τιμή),  postscript
           Για παράδειγμα:  set terminal postscript

  • set output "file name"
           Στέλνει την έξοδο του gnuplot στο file "file name". Η default τιμή είναι στην οθόνη.

  • Διεργασίες που αφορούν τους άξονες

  • set xrange [x-range]
               Προσδιορισμός του εύρους του x-άξονα για όλα τα γραφήματα. Η σύνταξη είναι ίδια με αυτή της εντολής plot. Πανομοιότυπα ισχύουν και για yrange, zrange. Αν δεν καθοριστεί το εύρος, τότε αυτό καθορίζεται αυτόματα από τα δεδομένα. ή ένα προκαθορισμένο διάστημα στον x-άξονα [-10:10] χρησιμοποιείται. Για να πάρετε το προκαθορισμένο διάστημα πληκτρολογήστε set xrange restore

  • set autoscale <option>
              Προσδιορίζει ότι όλη ή μέρος ενός διαστήματος προσδιορίζεται αυτόματα. Οι επιλογές είναι x, y, xmin, xmax, ymin, ymax και xy. Καμιά επιλογή ισοδυναμεί με xy. (Default είναι xy)

  • set [no]grid
              Σχεδιασμός ή όχι ενός πλέγματος πίσω από το γράφημα. (Default δεν σχεδιάζεται το grid)

  • set logscale <option>
             Προσδιορίζει λογαριθμικές κλίμακες. Οι επιλογές είναι x, y και xy. Default είναι μη λογαριθμική κλίμακα. Πληκτρολογώντας set logscale χωρίς επιλογές είναι ισοδύναμο με set logscale xy

  • set polar
            Θεωρεί τα δεδομένα σε πολικές συντεταγμένες

  • Επιλογή του μεγέθους του πλαισίου και του σχήματός του
    H εντολή set size επιτρέπει τον έλεγχο του μεγέθους και σχήματος του πλαισίου. Μερικές χρήσιμες εντολές είναι:

  • set size square
             Κατασκευάζει ένα τετράγωνο πλαίσιο, παρά αυτό που έχει το σχήμα του window

  • set size ratio r
             Προσπαθεί να ρυθμίσει το λόγο (ύψος/εύρος) του πλαισίου στη τιμή r.

  • set size noratio
              Επαναφέρει το προκαθορισμένο λόγο (ύψος/εύρος) του πλαισίου

  • set size x,y
              Προσδιορίζει τα x και y μεγέθη του πλαισίου σχετικά με το πλαίσιο που καταλαμβάνει όλο το window. Αν παραληφθεί το y, τότε λαμβάνεται x=y.

  • Δημιουργία ετικετών στους άξονες και σε άλλα σημεία

  • set xlabel "label"
              Δημιουργία ετικέτας για τον x-άξονα

  • set ylabel "label"
              Δημιουργία ετικέτας για τον y-άξονα

  • set x2label "label"
              Προσδιορισμός της ετικέτας που εμφανίζεται πάνω από τον ανώτερο x-άξονα.

  • set title "label"
              Δημιουργία ενός τίτλου για όλο το γράφημα. Εμφανίζεται στο πάνω μέρος

  • set label "label" at x, y
             Δημιουργία κειμένου στο καθορισμένο σημείο (σε μονάδες του γραφήματος)

  • set arrow from x, y to r, s
             Σχεδιασμός ενός βέλους μεταξύ των δεδομένων σημείων (σε μονάδες του γραφήματος)

  • set [no]key
             Δείχνει κάποιο σύμβολο για αναγνώριση ενός γραφήματος

  • Επιλογή του είδους της γραμμής και των σημείων
    Οι ακόλουθες εντολές εμφανίζονται στην εντολή plot και μετά από την with lines ή with points επιλογή. Προσδιορίζουν την εμφάνιση ή/και το μέγεθος των γραμμών και των συμβόλων που σχεδιάζονται

  • lt <option>
            Προσδιορίζει το τύπο της γραμμής -- το χρώμα της για το σκοπό μας. Επιλογή 1 δίνει κόκκινο, 2 πράσινο, 3 μπλε, 4 purple κλπ

  • lw <option>
           Προσδιορίζει το εύρος της γραμμής -- 1 είναι το προκαθορισμένο. Αυξανόμενοι αριθμοί δίνουν περισσότερο παχιές γραμμές

  • ps <option>
           Προσδιορίζει το μέγεθος του σημείου σχετικά με κάποιο προκαθορισμένο μέγεθος

  • pt <option>
           Προσδιορίζει το είδος του σημείου -- προσδιορίζει τον αριθμό των πλευρών ή προεξοχών
