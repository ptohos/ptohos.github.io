Προσαρμογή (fitting) δεδομένων μέσω του gnuplot  

Στα ακόλουθα θα δοθεί ένα παράδειγμα για το πως μπορούμε να χρησιμοποιήσουμε το gnuplot για να κάνουμε την προσαρμογή μιας καμπύλης στα δεδομένα μας. Θα υποθέσουμε ότι τα δεδομένα μας περιέχουν ένα πλήθος τυχαίων αριθμών κατανεμημένους σύμφωνα με Gaussian κατανομή

Τα δεδομένα αποτελούνται από τις τριπλέτες των αριθμών

Μπορείτε να δημιουργήσετε ένα script χρησιμοποιώντας ένα επεξεργαστή κειμένου όπως ο emacs ή μπορείτε να ζητήσετε από το gnuplot να κρατήσει σε ένα file όλες τις εντολές που πληκτρολογήσατε σε ένα interactive session. Για να το κάνετε αυτό δώστε την εντολή

       save "script-file-name"

Η εντολή γράφει όλες τις εντολές μέχρι αυτές που χρησιμοποιήθηκαν στο τελευταίο γράφημα σε ένα file με όνομα script-file-name.

Για να χρησιμοποιήσετε το script file αργότερα μπορείτε είτε να δώσετε το όνομα του file στην γραμμή εντολής που ξεκινά το gnuplot (gnuplot script-file-name) ή να δώσετε την εντολή:

       load "script-file-name"

ενώ βρίσκεστε μέσα σε περιβάλλον του gnuplot.

Ένα τυπικό παράδειγμα ενός script file είναι το ακόλουθο:

           set title "overall label"
           set xlabel "time"
           set ylabel "data"

           set nokey                   # απενεργοποιεί τη χρήση συμβόλων --- Παρατηρήστε ότι ότι ξεκινά με # είναι σχόλιο όπως εδώ

           plot [1:13] [-1:1]  "data1.dat" every 4 using 1:3 with points

Για να το τρέξετε δώστε την εντολή

            gnuplot -persist script-file-name      ή ενώ είστε μέσα στο gnuplot πληκτρολογήστε load "script-file-name"

Αν θέλετε μπορείτε να μετατρέψετε αυτό το script σε κάποιο executable το οποίο μπορείτε να τρέξετε χωρίς να χρειαστεί να πληκτρολογήσετε την εντολή gnuplot. Δηλαδή να το τρέξετε όπως τρέχετε τα προγράμματά σας μέχρι τώρα. Για να το κάνετε αυτό προσθέστε στην πρώτη γραμμή του script την εντολή:

            #! /usr/local/bin/gnuplot -persist

και κατόπιν μετατρέψτε το script file σε executable δίνοντας την εντολή Linux:  chmod  a+x  script-file-name.   H εντολή αυτή μετατρέπει το file σε executable το οποίο μπορούν να τρέξουν όλοι ανεξαιρέτως οι χρήστες.
Η πρώτη γραμμή που προσθέσαμε λέει στο λειτουργικό Linux πως να τρέξει το συγκεκριμένο script. Η προσέγγιση αυτή έχει αρκετά πλεονεκτήματα - για παράδειγμα μπορείτε να στείλετε τα δεδομένα ενός προγράμματος απευθείας στο script file και απλά να δώσετε στην εντολή plot την επιλογή "-" για όνομα του file δεδομένων, αποφεύγοντας έτσι να ξέρετε το όνομα του file από την αρχή.
