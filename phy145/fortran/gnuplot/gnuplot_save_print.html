Αποθήκευση αποτελεσμάτων και εκτύπωση

Ο καλύτερος τρόπος για να σώσετε τα γραφήματά σας είναι σε κάποιο Postscript file. Για να το κάνετε αυτό, χρησιμοποιήστε την εντολή

      set terminal postscript

ή αν επιθυμείτε μπορείτε να προσθέσετε κάποιες επιπλέον επιλογές. Για παράδειγμα:

      set terminal postscript color 18

θα δημιουργήσει ένα color Postscript με τις επικεφαλίδες ή ετικέτες των αξόνων με γραμματοσειρές μεγέθους 18.

Σύμφωνα με το προκαθορισμένο τρόπο του πακέτου gnuplot, το Postscript output θα πάει στην οθόνη το οποίο δεν θέλετε αν θέλετε να τυπώσετε ή να κρατήσετε το γράφημα αυτό σε κάποιο file. Για το λόγο αυτό θα πρέπει να διοχετεύσετε το output στο όνομα ενός file.

      set output "plotfile.ps"

Το ακρωνύμιο .ps  δεν είναι απαραίτητο αλλά χρησιμοποιείται για να ξέρουμε ότι το file είναι σε format postscript. Από τη στιγμή που το γράφημά σας έχει αποθηκευτεί στο file, μπορείτε να το στείλετε σε ένα εκτυπωτή χρησιμοποιώντας την εντολή Unix lpr ή να το δείτε στην οθόνη χρησιμοποιώντας την εντολή του Postscript viewer gv. Θα μπορούσατε τέλος να βάλετε το file αυτό σε κάποιο άλλο file εγγράφου.

Θα μπορούσατε να το στείλετε απευθείας στον εκτυπωτή μέσα από το gnuplot δίνοντας την εντολή:

      set output "|lpr"

Για να επιστρέψετε και πάλι στο να βλέπετε τα γραφήματά σας στην οθόνη θα πρέπει να θέσετε το terminal σε μορφή Χ11 δίνοντας την εντολή:

      set terminal X11
