Οδηγίες για την εγκατάσταση του πακέτου VPN

 

Η εγκατάσταση του πακέτου αυτού θα σας επιτρέψει να συνδεθείτε με τους υπολογιστές του εργαστηρίου της πληροφορικής όπου και να βρίσκεστε. Έτσι θα μπορέσετε να μεταφέρετε file σας (με ftp) ή αν έχετε εγκαταστήσει το λογισμικό cygwin σε windows PCs να συνδεθείτε απευθείας με τους υπολογιστές και να δουλέψετε εκεί.

  • Επισκεφθείτε την ιστοσελίδα του πανεπιστήμιου http://www.ucy.ac.cy/cc/noc/ucyvpn.doc

  • Ακολουθείστε τις οδηγίες για να κατεβάσετε το λογισμικό VPN ανάλογα με το λειτουργικό του υπολογιστή σας (Windows ή Macintosh)

  • Αφού κάνετε unzip το file που κατεβάσατε ακολουθήστε τις οδηγίες που περιέχονται στο file VPN_3000_Client.pdf για να κάνετε installation και  configuration του λογισμικού.

  • Ακολουθήστε τις οδηγίες για την εγκατάσταση του configuration file που σας επιτρέπει τη σύνδεσή σας με το πανεπιστήμιο. Ακριβώς με την ίδια διαδικασία μπορείτε να συνδεθείτε με τους υπολογιστές του εργαστηρίου της πληροφορικής κάνοντας import το αντίστοιχο configuration file. Στείλτε μου e-mail να σας δώσω το file αυτό.  

  • Επιλέξτε το όνομα του file όπως εμφανίζεται στο παράθυρο διαλόγου του VPN και κάντε δεξί click. Επιλέξτε  "Set as default Entry".

  • Για να συνδεθείτε κάνετε double click στο όνομα αυτό ή κάντε click στο εικονίδιο "Connect". Στο παράθυρο που θα ανοίξει θα σας ζητηθεί το username και password τα οποία είναι αυτά που έχετε στο εργαστήριο της πληροφορικής.

  • Αν δεν έχετε εγκαταστήσει το λογισμικό cygwin σε windows τότε μπορείτε να ανοίξετε ένα MS-DOS window (από το εικονίδιο Start στη κάτω αριστερά  γωνία επιλέξτε Run και στο παράθυρο πληκτρολογήστε cmd. Αυτό θα σας ανοίξει ένα μαύρο παράθυρο) και να πληκτρολογήσετε την εντολή ftp cs4054.in.cs.ucy.ac.cy. Θα σας ζητήσει το username και password τα οποία είναι αυτά που έχετε στα εργαστήρια της πληροφορικής. Δείτε τις οδηγίες για αντιγραφή files μεταξύ υπολογιστών στο link αυτό. 

  • Αν έχετε το λογισμικό cygwin τότε μπορείτε να κάνετε απευθείας login στους υπολογιστές του εργαστηρίου της πληροφορικής πληκτρολογώντας την εντολή ssh <username>@cs4054.in.cs.ucy.ac.cy. Θα σας ζητηθεί την πρώτη φορά να προσθέσετε το όνομα αυτού του υπολογιστή στη λίστα των known hosts (πληκτρολογήστε yes) και κατόπιν το password σας. Θα μπορούσατε να ακολουθήσετε επίσης τις οδηγίες για μεταφορά αρχείων μεταξύ υπολογιστών που βρίσκονται στο link αυτό.

  • Τα ονόματα των υπολογιστών του εργαστηρίου της πληροφορικής είναι της μορφής csXXXX.in.cs.ucy.ac.cy, όπου ΧΧΧΧ είναι ένα νούμερο που ξεκινά από 4030 και αυξάνει κατά 4 για τους διάφορους υπολογιστές. Σε οποιοδήποτε υπολογιστή συνδεθείτε μπορείτε να δείτε το home directory σας.

 

Είναι σημαντικό να ξέρετε πως δεν πρέπει να δώσετε το λογισμικό αυτό σε τρίτους που δεν συνδέονται με το πανεπιστήμιο και πως δεν πρέπει σε καμιά περίπτωση να φανερώσετε σε οποιονδήποτε το προσωπικό σας κωδικό πρόσβασης.
