• Οι παρακάτω οδηγίες αναφέρονται σε όσους έχουν κάποιο προσωπικό PC ή laptop το οποίο δουλεύει σε WINDOWS XP και θα ήθελαν να χρησιμοποιήσουν μέρος του πακέτου gnu για περισσότερη εξάσκηση πέρα από το εργαστήριο. Το πακέτο gnu είναι ελεύθερης πρόσβασης και περιέχει πολλά χρήσιμα προγράμματα και λογισμικά όπως τον FORTRAN compiler (g77), το πακέτο gnuplot που θα χρησιμοποιήσουμε, όπως επίσης και τον editor emacs. Όλα αυτά μπορείτε να τα κατεβάσετε από το διαδίκτυο. Όταν τελειώσετε θα έχετε ένα Linux-like περιβάλλον για WINDOWS. Ακολουθήστε τις παρακάτω οδηγίες:

  • Επισκεφτείτε την σελίδα http://www.cygwin.com/

  • Κάντε click στο σημείο που λέει Install or update now! (using setup.exe)

  • Στο παράθυρο που θα ανοίξει επιλέξτε Save

  • Θα σας ρωτήσει που θα θέλατε να σώσετε το setup.exe. Επιλέξτε ένα directory που θέλετε (π.χ. C:\Downloads) και πατήστε Save. To file θα αποθηκευτεί και ένα άλλο παράθυρο θα ανοίξει ζητώντας αν θέλετε να τρέξετε το setup.exe. Επιλέξτε run.

  • Στην επόμενη ερώτηση επιλέξτε Νext>. Στην επόμενη ερώτηση αφήστε το Default: Install from internet. To ίδιο και για την επόμενη ερώτηση. Αφήστε τις επιλογές όπως είναι (Root directory: C:\cygwin , Install for all users και Default Text File Unix). Στην ερώτηση για επιλογή Local Package Directory πληκτρολογήστε και πάλι C:\Downloads ή κάντε browse σ' αυτό το directory.

  • Αφήστε το default για το είδος της σύνδεσης (direct connection). Θα σας ρωτήσει από που θέλετε να κατεβάσετε τα αρχεία. Επιλέξτε μια σελίδα (π.χ. ftp://ftp.mirror.ac.uk) και πατήστε Νext>

  • H επόμενη ερώτηση είναι η πιο δύσκολη μια και πρέπει να επιλέξετε τα πακέτα που θα χρειαστεί να κατεβάσετε. Αν δεν επιλέξετε τίποτα και πατήσετε Νext θα κατεβάσει μόνο αυτά που βρίσκονται στην κατηγορία Base. Ωστόσο οι compilers, το gnuplot και οι editors δεν περιέχονται στο βασικό αυτό πακέτο. Για το λόγο αυτό θα χρειαστεί να κάνετε τις ακόλουθες επιλογές:

  • Πηγαίνετε το ποντίκι πάνω από τη λέξη Devel και κάντε click. Αυτό θα σας δείξει τα περιεχόμενα του πακέτου. Θα δείτε ότι τα περισσότερα λένε Skip. Κάνοντας μια φορά αριστερό click με το ποντίκι στη λέξη Skip των παρακάτω πακέτων επιλέγετε τα αντίστοιχα προγράμματα να αποθηκευτούν. Το ίδιο θα πρέπει να κάνετε και για τα υπόλοιπα πακέτα που φαίνονται στους παρακάτω πίνακες. Όταν τελειώσετε με την επιλογή πακέτων πατήστε Νext> και θα αρχίσει η διαδικασία. Μπορεί κατά την επιλογή σας να δείτε ότι κάποια από τα προγράμματα που σας αναφέρω βρίσκονται σε κατάσταση Keep ή είναι ήδη επιλεγμένα. Αφήστε τα όπως είναι. Επίσης όταν κάνετε επιλογή το πρόγραμμα θα πρέπει να δείχνει ένα x στο πρώτο τετράγωνο (κάτω από τη στήλη bi..).

  • Όταν τελειώσει η εγκατάσταση των προγραμμάτων θα δημιουργηθεί ένα shortcut με το όνομα Cygwin στο desktop. Κάνοντας double click θα πάρετε ένα καινούργιο παράθυρο σε bash shell. Δουλεύοντας σ' αυτό το παράθυρο είστε σε Linux περιβάλλον και ισχύουν όλες οι εντολές Linux. Μπορείτε να πληκτρολογήσετε xemacs και να πάρετε τον editor. Αν γράψετε ένα πρόγραμμα μπορείτε να το κάνετε compile με το γνωστό g77 -o <name>.x <name>.f και να το τρέξετε πληκτρολογώντας ./<name>.x.  

  • Για copy/paste λειτουργίες στο UNIX window μπορείτε να κάνετε δεξιό click με το ποντίκι ενώ βρίσκεστε στο συγκεκριμένο window. Επιλέξτε properties και στα Edit Options check (Quick Edit Mode) και uncheck (Insert Mode)

  • Aν θέλετε όλο το πακέτο χρειάζεστε περίπου ~350Mb χώρου.
