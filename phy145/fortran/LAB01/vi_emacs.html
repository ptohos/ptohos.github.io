ΣΚΟΠΟΣ

Στο αρχείο αυτό θα βρείτε μια σύντομη εισαγωγή για τους δυο περισσότερο δημοφιλείς Editors που χρησιμοποιούνται σε περιβάλλον Unix/Linux. Για κάθε Editor καλύπτονται τα ακόλουθα θέματα:

  • Σύντομη περιγραφή των επεξεργαστών κειμένου (vi, emacs)

  • Βασικές οδηγίες για εισαγωγή κειμένου και πλοήγηση (vi, emacs)

  • Μεταφορά και αντιγραφή κειμένου (vi, emacs)

  • Αναζήτηση και αντικατάσταση κειμένου (vi, emacs)

  • Άλλες χρήσιμες εντολές (vi, emacs)

  • Ένα πίνακα για γρήγορη αναζήτηση εντολών (vi, emacs)

Εισαγωγή στον editor vi

Ο vi είναι ένας editor που στηρίζεται σε κάποιο άλλο editor που ονομάζεται ex. Παρόλο που οι περισσότεροι αρχάριοι χρήστες βρίσκουν τον vi αρκετά μή φιλικό για να χρησιμοποιήσουν είναι ωστόσο πολύ χρήσιμο να μάθουν μερικές από τις βασικές λειτουργίες του αφού ο editor αυτός βρίσκεται σε όλα τα συστήματα τα οποία στηρίζονται σε Linux/Unix λειτουργικό σύστημα. Χρησιμοποιεί ακόμα καθιερωμένα πλήκτρα του πληκτρολογίου για εντολές και επομένως μπορεί να χρησιμοποιηθεί σε οποιαδήποτε πληκτρολόγιο ή workstation χωρίς να χρειάζεται να ανησυχούμε για μη συνήθεις αντιστοιχίες πλήκτρων. Ακόμα ο editor αυτός χρησιμοποιεί ελάχιστες πηγές (λειτουργίες) του λειτουργικού συστήματος με αποτέλεσμα να μπορούμε να το χρησιμοποιήσουμε ακόμα και όταν το σύστημα του υπολογιστή αντιμετωπίζει πολλά προβλήματα.

Για να ξεκινήσουμε με τον editor ανοίγουμε πρώτα ένα Linux terminal και πληκτρολογούμε την εντολή:

vi filename <RETURN>

όπου filename είναι το όνομα του αρχείου το οποίο θέλετε να επεξεργαστείτε. Αν το αρχείο δεν υπάρχει τότε ο editor vi θα το δημιουργήσει στο directory στον οποίο βρίσκεστε.

Βασική εισαγωγή κειμένου και πλοήγηση με τον editor vi

Το κύριο χαρακτηριστικό που κάνει τον vi μοναδικό στο είδος του είναι ότι ο δυαδικός τρόπος λειτουργίας: λειτουργία για εισαγωγή εντολών (command mode) και λειτουργία για εισαγωγή κειμένου (input mode). Όταν έχετε ενεργοποιήσει την λειτουργία εισαγωγής εντολών τότε οι χαρακτήρες που πληκτρολογούνται λαμβάνονται σαν εντολές και επομένως εκτελούν κάποια λειτουργίες (για παράδειγμα μετακινούν κείμενο, σβήνουν και αντιγράφουν κείμενο κλπ). Όταν έχετε ενεργοποιήσει την λειτουργία εισαγωγής κειμένου τότε οι χαρακτήρες που πληκτρολογείτε εισάγουν νέο κείμενο ή αντικαθιστούν υπάρχον κείμενο.

Όταν ξεκινάτε για πρώτη φορά το vi, αυτόματα είναι σε ισχύ η command mode. Για να μπείτε σε input mode πρέπει να πληκτρολογήσετε i (insert).   Μπορείτε τότε να πληκτρολογήσετε κείμενο το οποίο εισάγετε στην τρέχουσα θέση του cursor. Μπορείτε να διορθώσετε λάθη σας καθώς γράφετε με το πλήκτρο backspace. Για να επιστρέψετε και πάλι σε command mode πρέπει να πιέσετε το πλήκτρο ESC (πλήκτρο escape). Ένας άλλος τρόπος για εισάγετε κείμενο, ιδιαίτερα χρήσιμος όταν βρίσκεστε στο τέλος μιας γραμμής είναι να πιέσετε το πλήκτρο a (append).

Σε command mode, μπορείτε να κινείτε τον cursor μέσα στο κείμενό σας. Τα πλήκτρα h, j, k και l  κινούν το cursor αριστερά, κάτω, πάνω και δεξιά αντίστοιχα (αν είστε τυχεροί τα πλήκτρα με τα βέλη κατεύθυνσης μπορεί επίσης να κάνουν την ίδια λειτουργία). Άλλα χρήσιμα πλήκτρα είναι το ^ και το $ τα οποία σας μετακινούν απευθείας στην αρχή και στο τέλος μιας γραμμής, αντίστοιχα. Πιέζοντας το πλήκτρο w, πηγαίνετε  απευθείας στην αρχή της επόμενης λέξης και με το πλήκτρο b πηγαίνετε πίσω στην αρχή της προηγούμενης λέξης. Για να πάτε απευθείας στην αρχή του κειμένου σας πιέστε το πλήκτρο πρώτα το πλήκτρο 1 και μετά το G. Για να πάτε απευθείας στο τέλος του κειμένου σας τότε πιέστε μόνο το πλήκτρο G. Για να πάτε στην προηγούμενη σελίδα πιέστε το πλήκτρα ^B ενώ για να πάτε στην επόμενη σελίδα πιέστε ^F. Για να πάτε σε κάποια συγκεκριμένη γραμμή του κειμένου, πληκτρολογήστε τον αριθμό γραμμής και πιέστε G, π.χ. 55g θα σας πάει στη γραμμή 55.

Για να σβήσετε κείμενο, μετακινείστε τον cursor ακριβώς στο πρώτο χαρακτήρα του κειμένου που θέλετε να σβήσετε και σιγουρευτείτε ότι βρίσκεστε σε command mode. Πιέστε το πλήκτρο x για να σβήσετε το χαρακτήρα στον οποίο βρίσκεται ο cursor. Πληκτρολογώντας dw θα σβήσετε την επόμενη λέξη, d4w θα σβήσετε τις επόμενες 4 λέξεις ενώ πληκτρολογώντας dd θα σβήσετε την επόμενη γραμμή και 4dd τις επόμενες 4 γραμμές. Πληκτρολογώντας d$ μπορείτε να σβήσετε μέχρι το τέλος της γραμμής που βρίσκεστε ενώ dG θα σβήσει μέχρι το τέλος του κειμένου. Αν ανακαλύψετε ότι σβήσατε κατά λάθος αρκετό κείμενο, πιέζοντας το πλήκτρο u (undo) θα αποκαταστήσει την τελευταία αλλαγή.

Σε αρκετές περιπτώσεις θα θέλατε να ενώσετε 2 γραμμές σε μια. Πιέζοντας το πλήκτρο J μπορείτε να το πετύχετε (προσοχή πιέζοντας το πλήκτρο backspace δεν έχει το αποτέλεσμα το οποίο θα περιμένατε!).

Μετακινώντας και αντιγράφοντας κείμενο με τον vi

Ο vi χρησιμοποιεί buffers για να αποθηκεύσει το κείμενο το οποίο σβήνετε. Υπάρχουν 9 αριθμημένοι buffers (1-9) καθώς και ο undo buffer. Συνήθως ο buffer 1 περιέχει την πιο πρόσφατη αποκοπή κειμένου, ο 2 την επόμενη πρόσφατη κλπ.

Για να κόψετε (cut) και να επικολλήσετε (paste) κείμενο στο vi, πρώτα θα πρέπει να σβήσετε το κείμενο (χρησιμοποιώντας για παράδειγμα 5dd για να σβήσετε 5 γραμμές). Κατόπιν μετακινήστε το cursor στη γραμμή που θέλετε να επικολλήσετε το κείμενο που σβήσατε και πιέστε το πλήκτρο p (paste). Αν σβήσετε κάτι άλλο πριν επικολλήσετε, μπορείτε να επαναφέρετε το σβησμένο κείμενο κάνοντας paste τον αντίστοιχο buffer. Για να το κάνετε αυτό πληκτρολογείστε "1p,  "2p κλπ.

Για να αντιγράψετε (copy) και να επικολλήσετε (paste), τραβήξτε ("yank") το κείμενο που θέλετε (π.χ. 5yy για να αντιγράψετε 5 γραμμές). Κατόπιν μετακινήστε το cursor στη γραμμή που θέλετε να επικολλήσετε το κείμενο που αντιγράψατε και πιέστε p.

Αναζήτηση και αντικατάσταση κειμένου με τον vi

Ενώ βρίσκεστε σε command mode, μπορείτε να αναζητήσετε στο κείμενο για κάποια λέξη χρησιμοποιώντας μια συγκεκριμένη έκφραση. Για να ψάξετε το κείμενο που έπεται από τη θέση στην οποία βρίσκεστε πληκτρολογήστε / και κατόπιν μια συγκεκριμένη έκφραση και πατήστε <RETURN>. Για να ψάξετε στο κείμενο που βρίσκετε πριν από την τρέχουσα θέση του cursor, πληκτρολογήστε πρώτα ? αντί του /. Για να πάτε στην επόμενη έκφραση που πληροί τη έκφραση που ψάχνετε πιέστε n.  

Για να ψάξετε και να αντικαταστήσετε όλες τις περιπτώσεις κάποιου υπάρχοντος συνδυασμού (pattern1) με νέο συνδυασμό (pattern2), πληκτρολογείστε :%s/pattern1/pattern2/g και πιέστε <RETURN>.  Αν θέλετε να ερωτηθείτε για κάθε αλλαγή ώστε να την επιβεβαιώσετε προσθέστε το γράμμα c (confirm) στο τέλος της παραπάνω εντολής. Επίσης αντί του % μπορείτε να καθορίσετε ένα εύρος τιμών αριθμών γραμμής (π.χ. 1, 17) στο οποίο να λάβει χώρα η αντικατάσταση που θέλετε.

Άλλες χρήσιμες εντολές του vi (αποθήκευση, έξοδος από πρόγραμμα)

Προγραμματιστές μπορεί να βρουν χρήσιμη την εντολή :set number <RETURN> η οποία δείχνει τους αριθμούς γραμμών (η εντολή :set nonumber <RETURN> αναιρεί την προηγούμενη εντολή).

Για να σώσετε ένα αρχείο το οποίο επεξεργάζεστε πληκτρολογήστε :w <RETURN>. Για να σώσετε το αρχείο και να βγείτε από τον vi τότε πληκτρολογήστε την εντολή :w <RETURN> ή πληκτρολογήστε ZZ. Για να προκαλέσετε έξοδο από τον vi χωρίς να σώσετε το κείμενο πληκτρολογήστε την εντολή :q! <RETURN>.

Για να ξεκινήσετε να επεξεργάζεστε ένα άλλο αρχείο, πληκτρολογήστε :e <filename> <RETURN>.

Για να εκτελέσετε εντολές φλοιού (shell commands) μέσα από το vi, και να επιστρέψετε και πάλι στον vi μετά την εκτέλεση τους, πληκτρολογήστε την εντολή :!shellcommand <RETURN>.

Μπορείτε να χρησιμοποιήσετε το σύμβολο % σαν υποκατάστατο του ονόματος του αρχείου που επεξεργάζεστε (για παράδειγμα η εντολή :!echo % θα σας δείξει το όνομα το file που επεξεργάζεστε).

H εντολή . επαναλαμβάνει την τελευταία εντολή που δώσατε.

Εισαγωγή στον editor emacs

Ο emacs είναι ο πιο δημοφιλής επεξεργαστής κειμένου για συστήματα UNIX, Windows, και Macintosh. Αντίθετα με τον vi δεν είναι μέρος του λειτουργικού του UNIX αλλά μπορεί να αποκτηθεί από την Free Software Foundation.

Κάποιος φανατικός οπαδός του emacs θα σας πει ότι ο editor αυτός προσφέρει τρομερές ευκολίες που πηγαίνουν πολύ πιο μακριά από την απλή εισαγωγή και σβήσιμο κειμένου: μπορείτε να δείτε ταυτόχρονα περισσότερα από 1 αρχεία, να μεταφράσετε (compile), να τρέξετε (execute) και να βρείτε λάθη (debug) προγράμματα σε σχεδόν όλες τις γλώσσες προγραμματισμού, είδος κειμένων, να τρέξετε εντολές φλοιού, να διαβάσετε σελίδες από κείμενα οδηγιών, το e-mail σας, και ακόμα να ανατρέξετε στο διαδίκτυο μέσα από τον emacs. O editor αυτός είναι πολύ ευέλικτος αφού μπορείτε να επαναπροσδιορίσετε και να ορίσετε ποια πλήκτρα να εκτελούν συγκεκριμένες εντολές της αρεσκείας σας. Μπορείτε επίσης να γράψετε Lisp προγράμματα τα οποία προσθέτουν νέες εντολές και τρόπους εμφάνισης αφού ο emacs έχει το δικό του Lisp μεταφραστή. Και πραγματικά οι περισσότερες εντολές επεξεργασίας είναι γραμμένες ήδη σε Lisp. Μερικές εξαιρέσεις είναι γραμμένες σε C για καλύτερη απόδοση. Ωστόσο οι χρήστες δεν χρειάζονται να γνωρίζουν Lisp για να χρησιμοποιήσουν τον emacs.

Επικριτές του emacs θεωρούν ότι χρησιμοποιεί μεγάλο αριθμό από τις λειτουργίες του υπολογιστή σχετικά με τον vi και ότι έχει αρκετά πολύπλοκη δομή εντολών.

Στην πραγματικότητα οι περισσότεροι χρήστες χρησιμοποιούν και τους δυο editors - vi για γρήγορη επεξεργασία μικρών προγραμμάτων - και emacs για περισσότερο πολύπλοκες εργασίες οι οποίες απαιτούν τον έλεγχο περισσότερων του ενός αρχείων ταυτόχρονα.

Σε συστήματα UNIX, ο emacs μπορεί να τρέχει με γραφική υποστήριξη από το σύστημα Χ Windows στην οποία περίπτωση προσφέρονται κάποια χρήσιμα menu εντολών και αντιστοιχίας πλήκτρων και εντολών ή εντολών και πλήκτρων του mouse. Ωστόσο όλες οι ευκολίες αυτές προσφέρονται σε text μόνο τερματικά.

Για να ξεκινήσετε τον emacs ανοίξτε ένα terminal και πληκτρολογήστε:

                                          emacs <filename>  <RETURN>

όπου filename είναι το όνομα του file που θέλετε να επεξεργαστείτε. Αν το file δεν υπάρχει τότε θα δημιουργηθεί από τον emacs αυτόματα. Μπορείτε ακόμα να πληκτρολογήσετε την εντολή θέτοντας το σύμβολο & μετά το όνομα του file. Με το τρόπο αυτό o emacs "τρέχει" στο υπόβαθρο χωρίς να επηρεάζει την λειτουργία του terminal window από το οποίο ανοίξατε τον emacs. Αυτός είναι και ο πλέον ενδεικτικός τρόπος.

                                          emacs <filename> & <RETURN>

 

Βασικές λειτουργίες εισαγωγής και πλοήγησης με τον emacs

Η εισαγωγή κειμένου και πλοήγηση στον emacs επιτυγχάνεται κύρια με την χρήση των πλήκτρων διεύθυνσης για να τοποθετηθεί ο cursor στην επιθυμητή θέση και κατόπιν απευθείας πληκτρολόγηση του κειμένου που θέλετε. Για να χρησιμοποιηθούν περισσότερο πολύπλοκες εντολές χρειάζεται τις περισσότερες φορές να πιέσετε το πλήκτρο Ctrl (μερικές φορές φέρει το όνομα Control ή Ctl) ή το πλήκτρο Meta (μερικές φορές γράφεται με Alt. Αν τα πλήκτρα Meta και Alt δεν υπάρχουν τότε μπορείτε να χρησιμοποιήσετε το πλήκτρο ESC). Οι εντολές emacs περιγράφονται συνήθως ως ακολούθως:

  • C-<chr>  σημαίνει ότι κρατείτε πιεσμένο το πλήκτρο Ctrl καθώς πληκτρολογείτε το χαρακτήρα <chr>.
                    Επομένως, C-f σημαίνει κρατήστε πιεσμένο το πλήκτρο Ctrl και πληκτρολογείστε f.

  • M-<chr>  σημαίνει ότι κρατείτε πιεσμένο το πλήκτρο Meta ή Alt καθώς πληκτρολογείτε <chr>.
                    Αν έχετε μόνο το πλήκτρο ESC τότε πιέστε και αφήστε το αμέσως και πληκτρολογείστε <chr>

Μια αρχικά ενοχλητική ιδιαιτερότητα του emacs είναι ότι εντολή βοήθειας αντιστοιχεί στο συνδυασμό C-h (Ctrl h) στον οποίο αντιστοιχεί ο κώδικας για τη λειτουργία του backspace πλήκτρου στα περισσότερο συστήματα. Ωστόσο μπορείτε να αποφύγετε το πρόβλημα αυτό κάνοντας το πλήκτρο να δουλεύει όπως πρέπει δημιουργώντας στο home directory σας το αρχείο .emacs (το αρχείο αυτό διαβάζεται πάντοτε όταν δίνετε την εντολή emacs) και στο οποίο εισάγετε την ακόλουθη γραμμή:  

                               (global-set-key "\C-h" 'backward-delete-char-untabify)

Μπορείτε να κατεβάσετε ένα .emacs αρχείο από αυτό το link. Περιέχεται η παραπάνω εντολή καθώς και αρκετές άλλες χρήσιμες εντολές.

Για να αποκτήσετε πρόσβαση στο help menu μπορείτε να πληκτρολογήσετε M-x help <RETURN> (ή για κάποιο περισσότερο κατανοητό μάθημα για τη χρήση του emacs πληκτρολογήστε M-x help-with-tutorial <RETURN>)

Χρήσιμες εντολές πλοήγησης είναι οι C-a (αρχή της γραμμής), C-e (τέλος γραμμής), C-v (επόμενη σελίδα), M-v (προηγούμενη σελίδα), Μ-< (αρχή του file) και Μ-> (τέλος του file). Η εντολή C-d θα σβήσει το χαρακτήρα πάνω στον οποίο βρίσκεται ο cursor ενώ ο συνδυασμός C-k θα σβήσει μέχρι το τέλος της γραμμής. Κείμενο το οποίο έχει σβηστεί με το συνδυασμό C-k τοποθετείται σε buffer ο οποίος μπορεί αργότερα να αντιγραφεί με τον συνδυασμό C-y.  

Μεταφέροντας και αντιγράφοντας κείμενο στον emacs

Ο απλούστερος τρόπος για αποκοπή και επικόλληση κειμένου είναι να πάτε στην αρχή του κειμένου που θέλετε να σβήσετε και να πληκτρολογήσετε την εντολή C-k μέχρις ότου έχετε σβήσει όλο το κείμενο που θέλετε. Κατόπιν μετακινηθείτε στο σημείο που θέλετε να εισάγετε το κείμενο που μόλις σβήσατε και πληκτρολογήστε την εντολή C-y. Αν κάνατε κάποιο λάθος, ο συνδυασμός C-u, θα επαναφέρει το κείμενο που σβήσατε (ο emacs υποστηρίζει πολλά επίπεδα από undo και όχι μόνο της τελευταίας εντολής). Ένας άλλος τρόπος για να σβήσετε μεγάλο τμήμα κειμένου είναι να τοποθετήσετε το cursor στην αρχή του κειμένου και να πατήσετε C-SPC (SPC σημαίνει το πλήκτρο spacebar, αυτός ο συνδυασμός θέτει το αρχικό ορόσημο). Κατόπιν μετακινήστε το cursor στο τέλος του κειμένου που θέλετε να σβήσετε και πατήστε το συνδυασμό C-w. Μπορείτε να επαναφέρετε το κείμενο στην σωστή θέση με τον συνδυασμό C-y.

Για να αντιγράψετε και να επικολλήσετε, πρώτα σβήστε το κείμενο που θέσατε σα στόχο (όπως περιγράφτηκε στην προηγούμενη παράγραφο) και κατόπιν χρησιμοποιήστε το συνδυασμό C-y δύο φορές (μια για να επαναφέρετε το αρχικό κείμενο και μια για να δημιουργήσετε το αντίγραφο).

Αναζήτηση για κείμενο και αντικατάσταση κειμένου στον emacs

Για να διερευνήσετε την ύπαρξη κάποιας λέξης ή συνδυασμού χαρακτήρων στο κείμενο μπορείτε να χρησιμοποιήσετε το συνδυασμό C-s (για αναζήτηση προς τα κάτω της θέσης που βρίσκεστε) ή το συνδυασμό C-r για αναζήτηση προς τα πάνω. Πιέζοντας C-s ή C-r ξανά επαναλαμβάνει την αναζήτηση. Όταν έχετε βρει το κείμενο που θέλετε τότε πιέστε <RETURN> ή πιέστε το συνδυασμό C-g. Αυτός ο τελευταίος συνδυασμός ακυρώνει την εντολή και επιστρέφει τον cursor στην θέση στην οποία βρίσκονταν όταν ξεκίνησε η αναζήτηση.

Για να αντικαταστήσετε μια γραμματοσειρά, εισάγετε το συνδυασμό M-x replace-string <RETURN> (ίσως να θέλετε να τροποποιήσετε το .emacs file σας ώστε η εντολή αυτή να αντιστοιχεί στο συνδυασμό C-x r).  Ο συνδυασμός M-% εκτελεί αναζήτηση και αντικατάσταση μέχρι εξαντλήσεως όλων των περιπτώσεων (μπορείτε πάντοτε να το σταματήσετε).

Άλλες χρήσιμες εντολές του emacs

Εισάγοντας C-g θα τερματίσει την εντολή του emacs που βρίσκεται σε εξέλιξη.

Για να αποθηκεύσετε ένα file εισάγετε τον συνδυασμό C-x C-s. Για να ξεκινήσετε να επεξεργάζεστε ένα νέο file εισάγετε το συνδυασμό C-x C-f.

Για να ανοίξετε δυο windows (ή δυο buffers στη γλώσσα του emacs), εισάγετε το συνδυασμό C-x 2 (C-x 1 σας επιστρέφει και πάλι στο 1 window). O συνδυασμός C-o εναλλάσσει τον cursor μεταξύ των δυο windows στην ίδια οθόνη. Ο συνδυασμός C-x b σας επιτρέπει να αλλάξετε μεταξύ των ενεργών buffers ανεξάρτητα από το αν μπορείτε να τους δείτε ή όχι. Ο συνδυασμός C-x C-k καταστρέφει ένα buffer όταν δεν τον χρειάζεστε περισσότερο. 

Ο συνδυασμός C-x shell <RETURN> φέρνει ένα UNIX shell μέσα στο buffer (ίσως θα θέλατε να τροποποιήσετε το .emacs file και να θέσετε την εντολή αυτή σαν C-x C-u). Ο συνδυασμός M-x goto-line <RETURN> πηγαίνει σε μια συγκεκριμένη γραμμή (μπορείτε να το βάλετε σαν εντολή C-x g). Ο συνδυασμός M-x compile <RETURN> θα προσπαθήσει να κάνει compilation ενός προγράμματος (χρησιμοποιώντας την λειτουργία make ή κάποια άλλη εντολή που δίνετε). Αν γράφετε πολλά προγράμματα ίσως ήταν χρήσιμο να αντιστοιχίσετε  την εντολή αυτή στο συνδυασμό C-x c.

Ο συνδυασμός M-q θα αλλάξει τη forma μιας παραγράφου

Ο συνδυασμός C-x C-c σας βγάζει από τον emacs.

Γρήγορα κάρτα αναφοράς εντολών emacs

  • Μετακινήσεις του cursor:
    C-a                                  αρχή της γραμμής 
    C-e                                  τέλος της γραμμής
    C-<                                  αρχή του αρχείου
    C->                                  τέλος του αρχείου
    M-x goto-line                    πήγαινε στη γραμμή
    C-v                                   επόμενη σελίδα
    M-v                                  προηγούμενη σελίδα
     

  • Σβήσιμο και μετακίνηση κειμένου:
    Backspace                        σβήσιμο χαρακτήρα πριν το cursor (πρέπει να τροποποιηθεί το file .emacs)
    C-d                                   σβήσιμο του χαρακτήρα κάτω από τον cursor
    M-d                                  σβήσιμο λέξης
    C-k                                   σβήσιμο μέχρι το τέλος γραμμής (γραμμών). Επαναφορά με C-y
    C-SPC                              τοποθέτηση αρχικού ορόσημου
    C-w                                   σβήσιμο από το αρχικό ορόσημο μέχρι τη τρέχουσα θέση του cursor
     

  • Αναζήτηση και αντικατάσταση:
    C-s                                σταδιακή αναζήτηση προς τα εμπρός
    C-r                                    σταδιακή αναζήτηση προς τα πίσω
    C-%                                  πολλαπλή αντικατάσταση με αναζήτηση
    M-x replace-string <RETURN> <search string> <RETURN> <replace string> <RETURN>
     

  • Διάφορα:
    C-x u                             undo
    C-x C-s                             αποθήκευση αρχείου
    C-x C-f                              αναζήτηση αρχείου
    C-x 2                                 δημιουργία 2 windows
    C-x 1                                 δημιουργία 1 window
    C-x o                                 εναλλαγή μεταξύ windows
    C-x b                                 εναλλαγή μεταξύ buffers
    M-q                                   μετατροπή της φόρμας παραγράφου
    C-x C-c                             έξοδος από τον emacs
     

  • Χρήσιμες τροποποιημένες εντολές (στο αρχείο .emacs που σας δόθηκε)
    C-x g                             πήγαινε στη γραμμή
    C-x c                                 compilation προγράμματος (χρήσιμο με C-x 2)
    C-x C-u                             άνοιγμα ενός Unix shell (χρήσιμο με C-x 2)
    C-x a                                 επανάληψη εντολής
    C-x m                                μή αυτόματη εισαγωγή
