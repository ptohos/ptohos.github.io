Εξάσκηση με βασικές εντολές του λειτουργικού Linux

  1. Ανοίξτε ένα terminal window.

  2. Εισάγετε τις ακόλουθες εντολές και προσπαθήστε να εξηγήσετε το αποτέλεσμα που σας δίνουν. Ρωτήστε αν δεν καταλαβαίνετε και μη φοβάστε να πειραματιστείτε (σα κανονικός χρήστης δεν μπορείτε να προκαλέσετε οποιοδήποτε πρόβλημα):

    • echo hello world  <Return>

    • passwd <Return>

    • date <Return>

    • hostname <Return>

    • arch <Return>

    • uname -a <Return>

    • dmesg | more <Return>  (μπορεί να χρειαστεί να πιέσετε q για να βγείτε [quit])

    • uptime <Return>

    • who am i <Return>

    • who <Return>

    • id <Return>

    • last <Return>

    • finger <Return>

    • w  <Return>

    • top <Return>  (μπορεί να χρειαστεί να πιέσετε q για να βγείτε)

    • echo $SHELL <Return>

    • echo {con,pre}{sent,fer}{s,ed} <Return>

    • man "automatic door" <Return>

    • man ls <Return> (μπορεί να χρειαστεί να πιέσετε q για να βγείτε)

    • man who <Return> (μπορεί να χρειαστεί να πιέσετε q για να βγείτε)

    • lost <Return>

    • clear <Return>

    • cal 2000 <Return>

    • cal 9 1752 <Return> (Παρατηρείτε οτιδήποτε περίεργο?)

    • bc -l <Return> (μπορεί να χρειαστεί να πιέσετε q ή Ctrl-d για να βγείτε)

    • echo 5+4 | bc -l <Return>

    • yes please <Return> (μπορεί να χρειαστεί να πιέσετε Ctrl-c για να βγείτε)

    • time sleep 5 <Return>

    • history <Return>
