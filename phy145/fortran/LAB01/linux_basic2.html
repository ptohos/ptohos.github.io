  2. Συνεχίστε να διερευνάτε το δέντρο του filesystem χρησιμοποιώντας τις εντολές cd, ls, pwd, και cat. Κοιτάξτε στους directories /bin, /usr/bin/, /sbin, /tmp, και /boot. Τι παρατηρείτε?

  3. Εξετάστε το directory /dev. Μπορείτε να δείτε ποιές μονάδες (δίσκοι, floppies είναι στη διάθεσή σας?) Μπορείτε να βρείτε το terminal window που έχετε ανοίξει (tty)? Πληκτρολογήστε who am i μπορεί να βοηθήσει. Ποιός είναι ο κάτοχος του tty σας? (χρησιμοποιήστε ls -l)

  4. Διερευνήστε το directory /proc. Δείτε το περιεχόμενο των files interrupts, devices, cpuinfo, meminfo και uptime χρησιμοποιώντας την εντολή cat. Αυτός ο directory (/proc) είναι ένα ψευδο-filesystem που επιτρέπει να δούμε πληροφορίες σχετικά με τη δομή του λειτουργικού.

  18. Πηγαίνετε σε κάποιο directory που δεν κατέχετε (αποφύγετε καλού κακού τους subdirectories /proc και /dev) και προσπαθήστε να σβήσετε τα περιεχόμενά τους.

  19. Πειραματιστείτε με τις επιλογές της εντολής ls. Τι ακριβώς κάνουν οι επιλογές d, i, R και F?

  20. Δοκιμάστε τρεις διαφορετικούς τρόπους για να θέσετε τις άδειες πρόσβασης σε κάποιο file σε r--r--r--. Δημιουργήστε ένα file και δείτε αν δουλεύει.

  21. Πληκτρολογήστε umask 000 και κατόπιν δημιουργήστε ένα file world.txt. το οποίο περιέχει τις λέξεις "hello world". Κοιτάξτε στις άδειες πρόσβασης του file. Τι συνέβη ? Τώρα πληκτρολογήστε umask 22 και δημιουργήστε ένα file world2.txt. Πως μπορεί οι εντολές αυτές να είναι χρήσιμες?

  22. Δημιουργήστε ένα αρχείο "hello.txt" στο home directory σας χρησιμοποιώντας την εντολή cat -u > hello.txt. Ανοίξτε ένα άλλο terminal window και εκτελέστε την εντολή tail -f hello.txt. Τώρα στο αρχικό terminal window αρχίστε να πληκτρολογείτε κάποιο κείμενο. Τι εμφανίζεται στο δεύτερο terminal window?

  26.  Χρησιμοποιήστε το συνδυασμό των εντολών find, grep και sort για να δείτε όλα τα αρχεία τα οποία βρίσκονται στο /home subdirectory σας και περιέχουν την λέξη hello.

  27. Χρησιμοποιήστε την εντολή locate για να βρείτε όλα τα files τα οποία περιέχουν την λέξη emacs. Μπορείτε να συνδυάσετε την εντολή αυτή με την εντολή grep ώστε να μη βλέπετε τα ονόματα των files που περιέχουν την λέξη lib?
