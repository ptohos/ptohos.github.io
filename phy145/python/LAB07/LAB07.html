Δοκιμάστε τα ακόλουθα προγράμματα

  1. Να κάνετε τις γραφικές παραστάσεις για την λύση της εξίσωσης cos(x) = e-xsin(x) καθώς και της συνάρτησης f(x)=e-xsin(x) - cos(x) στις οποίες να φαίνεται η λύση τους [Προτεινόμενη λύση]

  2. Να βρείτε τη λύση της f(x) = e-x*xcos(4x) σύμφωνα με την μέθοδο brute force και να υπολογίσετε το σφάλμα από την αναμενόμενη τιμή της λύσης π/8. Βρείτε το σφάλμα για την περίπτωση 1001 και 10001 διαστημάτων.   [Προτεινόμενη λύση]

  3. Χρησιμοποιώντας τη μέθοδο βελτιστοποίησης brute force, να βρείτε τα μέγιστα και ελάχιστα τη συνάρτησης  f(x) =  e-x*xcos(4x) στο διάστημα [0,4]. [Προτεινόμενη λύση]

  4. Τροποποιήστε το πρόγραμμα naive_Newton ώστε να τυπώνει τις ενδιάμεσες λύσεις καθώς και τον αριθμό των επαναλήψεων που χρειάζονται για να επιτευχθεί η επιθυμητή λύση για την εξίσωση f(x) = x2 - 9 = 0. Πόσα βήματα θα χρειαστούν για ακρίβεια 10-15;  [Προτεινόμενη λύση]

  5. Χρησιμοποιώντας το πρόγραμμα της άσκησης 4, βρείτε τη λύση της εξίσωσης f(x) = tanh(x) = 0. H λύση της εξίσωσης είναι x=0. Θεωρήστε ακρίβεια ε=0.001 και αρχική τιμή πρόβλεψης x0 = 1.08. Κάντε το γράφημα της  εφαπτομένης σε κάθε επανάληψη της μεθόδου.  Επαναλάβετε την ίδια διαδικασία (βήματα επίλυσης και γράφημα της εφαπτομένης) για x0 = 1.09. Yπενθύμιση: tanh(x) = (ex - e-x)/(ex + e-x) = (e2x - 1)/(e2x + 1) [Προτεινόμενη λύση]

  6. Εφαρμόστε την μέθοδο Secant για την εύρεση της λύσης της εξίσωσης  f(x) = x2 - 9 = 0. Θεωρήστε ως αρχικά σημεία x0 = 1000 και x1 = x0 - 1. [Προτεινόμενη λύση].

  7. Χρησιμοποιήστε την μέθοδο bisection για την συνάρτηση της άσκησης 5 και αρχικό διάστημα [-5,3]. [Προτεινόμενη λύση]
