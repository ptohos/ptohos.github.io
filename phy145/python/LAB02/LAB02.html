ΕΞΑΣΚΗΣΗ

  • Κάνετε logon στο σύστημα. Δημιουργήστε ένα καινούριο directory στο home directory σας και ονομάστε τον LAB02. Θα πρέπει να πάτε στο νέο αυτό directory και ότι προγράμματα γράψετε στο σημερινό εργαστήριο θα πρέπει να βρίσκονται σε αυτό τον directory.

  • Για να εισέλθετε σε περιβάλον python, δώστε στο terminal την εντολή python3. Μπορείτε στο περιβάλλον αυτό να πληκτρολογήσετε διάφορες εντολές python. Αν το πρόγραμμά σας γίνεται πιο μεγάλο, μπορείτε να γράψετε τις εντολές σε ένα αρχείο στο οποίο δίνετε το ακρόνυμο py. Μπορείτε ενώ είστε σε περιβάλλον python να δώσετε την εντολή import program name ή αν βρίσκεστε στο terminal prompt να δώσετε την εντολή python3 <program_name>.py όπου program_name ειναι το όνομα του προγράμματος που δημιουργήσατε.  Τo πρόγραμμά σας το πληκτρολογείτε με την χρήση του editor emacs. 

  • Κατεβάστε το πρόγραμμα constvar.py . Για να το κατεβάσετε κάντε click με το ποντίκι πάνω στo link και μετά save. Το πρόγραμμα αυτό σας ζητά 3 ακέραιους και 3 πραγματικούς αριθμούς καθώς και 2 μεταβλητές χαρακτήρων. Κατόπιν αφού τυπώσει τις τιμές που δώσατε, θα εκτελέσει κάποιες απλές πράξεις και θα τυπώσει τα αποτελέσματα. Ανοίξτε το file και δείτε τις εντολές του προγράμματος.

  1. Με βάση το προηγούμενο πρόγραμμα, γράψτε τώρα ένα δικό σας το οποίο δέχεται  4 πραγματικούς αριθμούς (a,b,c,d) και 2 ακέραιους (i,j) από το πληκτρολόγιο και κατόπιν τυπώνει στην οθόνη τα αποτελέσματα των ακόλουθων πράξεων: (α) z = (a+b)/(c+d)   (b) z = a+b/(c+d)   (c) z = (a+b)/c + d, (d) k = i/j και (e) z = 4**(i/j).  Δοκιμάστε να δώσετε τις ακόλουθες τιμές:  α = 10, β = 3.4, c = 12, d = 2, i = 1 και j = 2. [Προτεινόμενη λύση]

  2. Δοκιμάστε να γράψετε ένα πρόγραμμα που να υπολογίζει την τιμή 2**3**2 και την τιμή sin(π/2). [Προτεινόμενη λύση]

  3. Γράψτε ένα πρόγραμμα που να κάνει το ακόλουθα: Nα διαβάζει 2 πραγματικούς αριθμούς. Υπολογίστε την διαφορά μεταξύ του πρώτου αριθμού στο τετράγωνο και του δεύτερου στο τετράγωνο. Γράψτε τα 2 νούμερα, τα τετράγωνά τους και την διαφορά τους. [Προτεινόμενη λύση]

  4. Γράψτε ένα πρόγραμμα που να μετατρέπει βαθμούς Κελσίου σε Φαρεναϊτ και τους Φαρενάϊτ πάλι σε Κελσίου και να τυπώνει στην οθόνη τις δύο θερμοκρασίες. Η σχέση που συνδέει τις δύο κλίμακες θερμοκρασίας είναι: degF = (9/5)*degC + 32. Αν το πρόγραμμά σας είναι σωστό τότε θα πρέπει να πάρετε στην δεύτερη μετατροπή την θερμοκρασία που δώσατε αρχικά. [Προτεινόμενη λύση]
  1. Nα γράψετε ένα πρόγραμμα που να διαβάζει τρεις τυχαίους αριθμούς Α, Β, C και να τυπώνει το μεγαλύτερο τους. Το πρόγραμμα θα πρέπει κατόπιν να τυπώνει τους αριθμούς σε φθίνουσα σειρά. [Προτεινόμενη λύση]

  2.  Γράψτε ένα πρόγραμμα που να υπολογίζει και να τυπώνει το μέσο όρο Ν αριθμών οι οποίοι δίνονται διαδοχικά στον Η-Υ από το πληκτρολόγιο. Θα πρέπει να δίνετε το πλήθος των αριθμών, Ν,  από το πληκτρολόγιο. Το πρόγραμμα θα πρέπει να ζητά κάθε i-οστό αριθμό από το χρήστη και να  τυπώνει στην οθόνη τον αριθμό καθώς και την αύξουσα σειρά του πριν ζητήσει τον επόμενο αριθμό. [Προτεινόμενη λύση]

  3. Γράψτε ένα πρόγραμμα το οποίο δέχεται δύο ακέραιους αριθμούς από το πληκτρολόγιο και υπολογίζει το άθροισμα των ακεραίων που περιέχονται μεταξύ των δύο αυτών ακεραίων (ανοικτό διάστημα) καθώς και το μέσο όρο τους. Το πρόγραμμα θα πρέπει να ελέγχει αν το διάστημα που δίνεται είναι σωστό (δηλαδή αν το πάνω όριο είναι όντως μεγαλύτερο του κάτω ορίου και να ζητά από το χρήστη να διορθώσει την εισαγωγή στοιχείων χωρίς να σταματά) και αν υπάρχουν τα απαραίτητα στοιχεία ώστε να υπολογιστεί ο μέσος όρος.  Το πρόγραμμά σας θα πρέπει να τυπώνει το διάστημα, το πλήθος των ακεραίων που βρέθηκαν στο διάστημα αυτό, το άθροισμά τους και το μέσο όρο τους. [Προτεινόμενη λύση]

  4. Να γράψετε ένα πρόγραμμα που να υπολογίζει το παραγοντικό ενός ακέραιου αριθμού Ν που δίνεται από το χρήστη και να τυπώνει το παραγοντικό καθώς και τον αριθμό Ν. Ο ορισμός του παραγοντικού είναι: n! = 1·2·3···n, για n≥1. Για n=0, n! =1. [Προτεινόμενη λύση]
