ΦΥΣ 145: Υπολογιστικές Μέθοδοι στη Φυσική













 

 

Περιγραφή του μαθήματος:

Εισαγωγή σε βασικές μεθόδους αριθμητικής ανάλυσης και  εφαρμογές τους στη λύση προβλημάτων φυσικής και  ανάλυσης δεδομένων. Παράλληλη εκμάθηση της γλώσσας  προγραμματισμού Python και του λειτουργικού Linux/Unix.

Ώρες και χώρος διδασκαλίας:

 Διαλέξεις: κάθε Δευτέρα  11:30-13:00 στο εργαστήριο υπολογιστών του Τμήματος (αίθουσα 012 του ΘΕΕ02)

 Εργαστήρια: κάθε Δευτέρα 14:30-18:00 στο εργαστήριο υπολογιστών του Τήματος (αίθουσα 012 του ΘΕΕ02).


Αξιολόγηση/Bαθμολόγηση:

  15% quiz σε κάθε εργαστήριο

  15% ασκήσεις κατ’ οίκον

  30% μια ενδιάμεση πρόοδο

  40% τελική εξέταση


Εργαστήρια:

Τα εργαστήρια είναι υποχρεωτικά. Περισσότερες από μια απουσίες χωρίς απόλυτα σοβαρό λόγο ισοδυναμεί με αποτυχία στο μάθημα. 

Οι ασκήσεις των εργαστηρίων θα πρέπει να ολοκληρώνονται στη διάρκεια του εργαστηρίου ώστε να συζητηθούν στο τέλος κάθε εργαστηρίου.

Θα πρέπει να συμμετάσχετε ενεργά στις ασκήσεις των εργαστηρίων και να προσπαθείτε να λύσετε τις απορίες σας. Θα χωριστείτε σε ομάδες ώστε να είναι πιο εύκολη η διατύπωση και επίλυση ερωτήσεων.


Εργασίες κατ'οίκον:

Οι ασκήσεις για το σπίτι θα σας ανακοινώνονται μέσω του διαδικτύου στο τέλος κάθε εργαστηρίου και θα πρέπει να τις επιστρέφετε μέσω e-mail (fotis@ucy.ac.cy) μια εβδομάδα αργότερα πριν την αρχή του εργαστηρίου


Quiz

Σε κάθε εργαστήριο θα υπάρχει ένα 20-λεπτο quiz το οποίο θα στηρίζετε στην ύλη του προηγούμενου εργαστηρίου και θα αποτελείτε από ερωτήσεις πολλαπλής επιλογής ή θα σας ζητηθεί να γράψετε κάποιο μέρος προγράμματος. Το quiz θα γίνεται σε τυχαία ώρα κατά την διάρκεια του εργαστηρίου και θα πρέπει να το επιστρέφετε μέσω e-mail (fotis@ucy.ac.cy).


Computer Accounts:

Όσοι δεν έχετε  accounts στους υπολογιστές του εργαστηρίου του Τμήματος ή δεν θυμάστε τον κωδικό πρόσβασης στους υπολογιστές θα πρέπει να με ενημερώσετε άμεσα ώστε να διευθετηθεί η πρόσβασή σας στους υπολογιστές του εργαστηρίου .
