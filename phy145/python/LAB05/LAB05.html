  2. Γράψτε ένα πρόγραμμα για την συνάρτηση greet() του πρώτου προβλήματος, που δέχεται ένα μεταβλητό αριθμό ορισμάτων για το όνομα που θα τυπώσει δίπλα στο msg που περιέχει η συνάρτηση. Το msg αποτελεί το πρώτο όρισμα της συνάρτησης
    [Προτεινόμενη λύση]

  3. Να γράψετε μια συνάρτηση που δέχεται πολλαπλό αριθμό ορισμάτων που είναι ακέραιοι αριθμοί και υπολογίζει τη μέση τιμής τους και επιστρέφει το πλήθος των αριθμών αυτών.  [Προτεινόμενη λύση]

  4. Να γράψετε τo πρόγραμμα της class MyVector και της παράγωγης class  MyPolarVector που συζητήθηκε στην 5η διάλεξη.  Δοκιμάστε να τρέξετε τις εντολές από την διάλεξη. Προσθέστε επιπλέον συναρτήσεις/μεθόδους στην κλάση MyVector που να υπολογίζουν το εσωτερικό γινόμενο δύο διανυσμάτων, το εξωτερικό γινόμενο δύο διανυσμάτων, το γινόμενο και τη διαίρεση ενός διαλύσματος με ένα βαθμωτό μέγεθος. [Προτεινόμενη λύση]

  5. Να γράψετε μια class MyComplex για μιγαδικούς αριθμούς. Η κλάση θα πρέπει να δέχεται δύ float αριθμούς για το πραγματικό και φανταστικό μέρος του μιγαδικού αριθμού. Η συνάρτηση του constructor θα πρέπει να οριστεί με τέτοιο τρόπο ώστε να δέχεται τα δύο ορίσματα με default τιμή 0. Η κλάση θα πρέπει να έχει μια μέθοδο που διαβάζει το πραγματικό και φανταστικό μέρος που δίνεται όταν ορίζεται ένας μιγαδικός αριθμός και επιστρέφει το μιγαδικό σε μορφή a+jb. Θα πρέπει να έχετε επίσης μεθόδους που εφαρμόζουν τις βασικές πράξεις των μιγαδικών αριθμών (πρόσθαιση, αφαίρεση, πολ/σμο, διαίρεση, μέτρο). [Προτεινόμενη λύση]

  6. Να γράψετε το πρόγραμμα μιας παράγωγης κλάσης που ονομάζεται Triangle η οποία κληρονομεί τις ιδιότητες μια κλάσης Polygon που περιγράφει ένα κλειστό σχήμα από τρεις ή περισσότερες γραμμές.  Η βασική class κρατά δεδομένα για τον αριθμό των πλευρών του πολυγώνου και το μήκος της κάθε πλευράς που αποθηκεύεται σε μορφή μιας listας sides.  Υπάρχει ακόμα μια μέθοδος inpSides() που ζητά να εισαχθούν τα μήκη της κάθε πλευράς και μια μέθοδος dispSides() που τυπώνει τα μήκη των πλευρών. Το τρίγωνο είναι ένα πολύγωνο με 3 πλευρές. Όλες οι ιδιότητες της κλάσης polygon κληρονομούνται αυτούσιες από την κλάση Polygon. Δημιουργήστε μια νέα μέθοδο για την Triangle class που να υπολογίζει το εμβαδό Area() του τριγώνου.  [Προτεινόμενη λύση]
