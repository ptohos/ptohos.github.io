Πληροφορίες
 
       
     


Εδώ θα βρείτε ανακοινώσεις σχετικά με το μάθημα, όπως αλλαγές σε κάποιες ώρες, διορθώσεις, ώρες εξετάσεων και οτιδήποτε έχει σχέση με το μάθημα. Επίσης απαντήσεις σε απορίες συναδέλφων σας μπορεί να φανούν χρήσιμες σε όλους και θα συμπεριλαμβάνονται στη σελίδα αυτή.



(α) Για να εγκαταστήσετε σε περιβάλον MS Windows, virtualbox και ubuntu ώστε να έχετε πλήρη λειτουργικότητα Linux περιβάλλοντος μέσα σε Windows θα χρειαστεί η διαδικασία που περιγράφεται στο αρχείο αυτό.

(β) Μπορείτε να κατεβάσετε επίσης το αρχείο .emacs για να έχετε τον επεξεργαστή γλώσσας με χρώματα κλπ καθώς επίσης και το αρχείο .Xdefaults ώστε το terminal window να έχει τα σωστά settings. Θα πρέπει να κατεβάσετε τα δυο αυτά files χρησιμοποιώντας το mozilla browser μέσα από το virtual machine των ubuntu που έχετε εγκαταστήσει προηγουμένως.

(γ) Για να εγκαταστήσετε το λογισμικό ROOT (version v6-16-00) στο virtual machine του ubuntu κατεβάστε το script Root_git_ubuntu.sh και κάντε το copy στο home directory σας (σε terminal window dώστε την εντολή mv ~/Downloads/Root_git_ubuntu.sh ~/.). Κάντε το executable δίνοντας την εντολή chmod +x Root_git_ubuntu.sh σε ένα terminal window και κατόπιν δώστε την εντολή source Root_git_ubuntu.sh  Θα σας ζητήσει αρχικά το πassword sας για το installation και θα πρέπει να δώσετε μερικά Y για τα λογισμικά που θα εγκατασταθούν. Αν όλα πάνε καλά θα πρέπει να δείτε ένα μύνημα ότι η εγκατάσταση του Root ήταν επιτυχής.
Μετά την επιτυχή εγκατάσταση θα πρέπει να δώσετε τώρα την εντολή:  source /usr/local/Codes/root/v6-16-00/bin/thisroot.sh
Την εντολή αυτή θα πρέπει να την βάλετε και στο αρχείο .bashrc που βρίσκεται στο home directory σας ώστε να εκτελείτε κάθε φορά που ανοίγετε ένα τerminal window. Μετά την εντολή αυτή δώστε την εντολή root gια να μπείτε στο root.

Το παραπάνω script δουλεύει για την εγκατάσταση του του ROOT version v6-16-00. Θα μπορούσατε να αλλάξετε το version αλλάζοντας την τιμή της μεταβλητής RootVersion στο script με την επιθυμητή τιμή.

Σημείωση: μερικές φορές όταν κατεβάζετε το παραπάνω script προστίθενται επιπλέον χαρακτήρες που δεν εμφανίζονται στην οθόνη και δεν επιτρέπουν την εκτέλεσή του. Για να λυθεί το πρόβλημα αυτό χρειάζεται να μετατραπεί τo file από DOS σε Linux. Για να γίνει αυτό κάντε edit το file με τον emacs και δώστε την ακόλουθη εντολή <control>-x και πατήστε RET και μετά f (στην command line του emacs θα εμφανιστεί coding system for saving file:). Πληκτρολογήστε στη γραμμή αυτή unix.
