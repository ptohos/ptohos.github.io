Στο εργαστήριο γράψαμε ένα πρόγραμμα που διαβάζει δεδομένα από κάποιο αρχείο και δημιουργεί histograms και ntuple. Μπορείτε να κατεβάσετε το ReadFromFile.C καθώς και το αρχείο myfile.dat που θα πρέπει να διαβάσετε για να τρέξετε το συγκεκριμένο προγραμμα. Για να το τρέξετε θα πρέπει να δώσετε την εντολή .x ReadFromFile.C μέσα στο root.  Μπορείτε να δείτε τα histograms και να κάνετε καλύτερα γραφήματα χρησιμοποιώντας το file RootPlot.C. Aνοίξτε το για να δείτε τι έχει και τις εντολές που χρησιμοποιούνται. Για να το τρέξετε δώστε την εντολή .x RootPlot.C

  1. Να γράψετε ένα πρόγραμμα το οποίο λύνει μια διαφορική εξίσωση πρώτης τάξης με τη μέθοδο Euler. Σύμφωνα με τη μέθοδο Euler θα πρέπει να δοθούν οι αρχικές συνθήκες του προβλήματος (xo kαι yo), το εύρος του διαστήματος Δx και η παράγωγος του y ως προς x για να υπολογισθεί η νέα τιμή του y_(i+1) = y_i+dy/dx*Δx και να ανανεωθεί παράλληλα το x_(i+1) = x_(i) + Δx. Θεωρήστε ότι μελετάτε την κίνηση ενός σώματος που κινείται με σταθερή ταχύτητα και θέλετε την θέση του σώματος συναρτήσει του χρόνου για 100 βήματα Δt=0.1sec.

  1. Στην άσκηση αυτή θα χρησιμοποιήσετε τους τρόπους input/output που συζητήσαμε. Γράψτε ένα πρόγραμμα το οποίο τυπώνει ένα πίνακα δύο στηλών που δείχνει πόσες φορές εμφανίζεται κάθε γράμμα από το α ως το z στο file που περιέχει το πρόγραμμά σας. Δηλαδή θα πρέπει, αν σώσατε το πρόγραμμά σας με όνομα askisi2.cpp, θα πρέπει να το διαβάσετε όταν τρέχει το πρόγραμμα και να βρείτε πόσες φορές εμφανίζεται κάθε γράμμα του αλφαβήτου. Θεωρήστε ότι το πρόγραμμά σας έχει μόνο μια μεταβλητή τύπου ifstream, μια μεταβλητή τύπου char, και δυο μεταβλητές τύπου int. To αποτέλεσμα του προγράμματός σας θα πρέπει να μοιάζει όπως το παρακάτω:

             b                                3
             c                                4

  1. Στο πρόβλημα αυτό θα διαβάσετε στοιχεία από ένα file και θα πρέπει να χρησιμοποιήσετε  δομές struct για να κατηγοροποιήσετε διάφορα στοιχεία. Το file ονομάζεται Particles.dat και μπορείτε να το κατεβάσετε από την ιστοσελίδα του μαθήματος σε αυτό το link.
    H πρώτη στήλη του file περιέχει το event number που καταγράφηκε το σωματίδιο, η 2η στήλη περιέχει το id του σωματιδίου (το είδος του δηλαδή). Η σύμβαση είναι |id|=11 ηλεκτρόνια/ποζιτρόνια, 13 μιόνια/αντι-μιόνια. Σωματίδια με id τα οποία είναι έχουν αριθμούς πολ/σια του 4 ή του 5
    (π.χ. 411 ή 511 ή 10410, 10520, 5001) είναι αδρόνια. Οι επόμενες στήλες περιέχουν κατά σειρά την Ενέργεια, px, py, pz και μάζα του σωματιδίου.
    Στο πρόγραμμα σας θα πρέπει να διαβάσετε κάθε γραμμή και να κατηγοροποιήσετε τα σωματίδια σε collections από ηλεκτρόνια, μιόνια και αδρόνια ανάλογα με το είδος τους.
    Θα πρέπει η κάθε δομή να περιέχει σα μέλη, την Ενεργεια, px, py, pz, μάζα, φορτίο (προέρχεται από το id +,-) και το id του σωματιδίου. Χρησιμοποιώντας τα collections αυτά θα πρέπει να βρίσκετε για κάθε γεγονός ποιο από τα σωματίδια της συλλογής έχει εγκάρσια ορμή  μεγαλύτερη από GeV/c και να σχηματίσετε ένα νέο collection που είναι το collection με τα επιλεγμένα σωματίδια διαφορετικά για κάθε τύπο. Το πρόγραμμά σας θα πρέπει να τυπώνει στην οθόνη τον αριθμό των μιονίων που έχουν εγκάρσια ορμή μεγαλύτερη από 20 GeV/c kαι για κάθε ένα από αυτά να τυπώνει σε μία γραμμή τις τρεις συνιστώσες της ορμής του, το φορτίο του και την ποσότητα y= -0.5ln[(E+pz)/(E-pz)] που ονομάζεται ωκύτητα ή rapidity. Μην ξεχνάται να μηδενίζεται τα στοιχεία της δομής για κάθε γεγονός που που διαβάζετε.
