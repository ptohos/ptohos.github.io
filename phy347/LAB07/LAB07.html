Εργαστήριο 07

 
       
     

 


Στο εργαστήριο αυτό θα εξασκηθείται με την δημιουργία και γραφή κλάσεων.

  1. Έχετε μάθει στην Υπολογιστική Φυσική Ι πως να λύνετε κανονικές διαφορικές εξισώσεις (ODE) με τις μεθόδους Euler, Euler-Cromer, Runge-Kutta 2ou βαθμού και ίσως Runge-Kutta 4ou βαθμού. Στην άσκηση αυτή θα πρέπει να γράψετε μια κλάση που ονομάζεται ODE η οποία έχει σαν member συναρτήσεις τις μεθόδους επίλυσης διαφορικών εξισώσεων που προαναφέραμε.
    H μέθοδος Euler ουσιαστικά εκτελεί :   y(t(k+1)) = y(t=tk) + h*f(tk,yk)
    H μεθοδος Euler-Cromer εκτελεί:       y(t(k+1)) = y(t=tk) + h*f(tk+1, yk+1)
    H μέθοοδος Runge-Kutta 2ου βαθμού:   y(t(k+1)) = y(tk) + (F1 + F2)/2
      όπου F1 = h*f(tk,yk) και F2 = h*f(tk+1,yk + F1)
    H μέθοδος Runge_Kutta 4ου βαθμού:    y(t(k+1)) = y(tk) + (F1+2F2+2F3+F4)/6
      όπου F1=h*f(tk,yk) F2=h*f(tk+1/2h, yk+F1/2) F3=h*f(tk+1/2h,yk+F2/2) F4=h*f(tk+1,yk+F3)

    Στα παραπάνω f είναι η παράγωγος της συνάρτησης. Δοκιμάστε το πρόγραμμα σας για f=dx/dt=[(1-et)x]/(1+et)  για x(0)=3 και 0 < t =< 2

    Η ακριβής λύση είναι x(t) = 12et/(1+et)2

    Για την περίπτωση που θα χρειαστείται να δηλώσετε πίνακες μεταβλητού μεγέθους αυτό μπορείτε να το κάνετε με δυναμικό τρόπο (dynamic αllocation) ως εξής:
    int *array;
    array = new int[size];   Δημιουργείτε έτσι ένας πίνακας int με μέγεθος size
    Η μνήμη που χρησιμοποιείται μπορεί στο τέλος να ελευθερωθεί δίνοντας την εντολή delete [] array;

    Συγκρίνετε το πρόγραμμά σας με το lab07_prob.cpp


  2. Να γραφεί μία κλάση η οποία περιέχει όλες τις μεθόδους λύσης κανονικών διαφορικών εξισώσεων (Euler, Euler-Crommer, Runge-Kutta 2ου και 4ου βαθμού). Το πρόγραμμά σας θα πρέπει να περιέχει τον κατάλληλο constructor, καθώς επίσης και συναρτήσεις members της κλάσης για την συνάρτηση της παραγώγου ή της δεύτερης παραγώγου. Στον constructor θα πρέπει να περνάτε τις αρχικές συνθήκες του προβλήματος που ορίζουν κατάλληλα private data μέλη της κλάσης. Το πρόγραμμά σας θα πρέπει να περιέχει κλίσεις σε συναρτήσεις των κλάσεων του πακέτου Root  για κατασκευή γραφημάτων τα οποία θα πρέπει να σώσετε σε κάποιο file για μεταγενέστερη επεξεργασία. Θα μπορούσατε επίσης να ορίσετε ένα Root Tree με branches τις διάφορες συναρτήσεις και leaves τις μεταβλητές του προβλήματος. Θα πρέπει να εφαρμόσετε το πρόγραμμά σας για την περίπτωση ενός αρμονικού ταλαντωτή.

    Συγκρίνετε το πρόγραμμά σας με το     lab07_ode.cpp

Για να τρέξετε τα παραπάνω προγράμματα έξω από το λογισμικό root θα πρέπει πρώτα να τα κάνετε compile χρησιμοποιώντας το script CompileWithRoot.sh. Αρχικά θα πρέπει να κάνετε το file αυτό executable δίνοντας την εντολή chmod +x CompileWithRoot.sh
Μπορείτε κατόπιν να το τρέξετε δίνοντας την εντολή ./CοmpileWithRoot.sh ακολουθoύμενη από το όνομα του file που θέλετε να κάνετε compile και το όνομα του executable που θέλετε να δημιουργήσετε. Για παράδειγμα θα γράφατε: ./CompileWithRoot.sh test.cpp test.x όπου test.cpp είναι το file του κώδικά σας και test.x το executable που θέλετε να δημιουργήσετε.
