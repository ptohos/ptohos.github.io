    Θα πρέπει να αποθηκεύσετε το αρχείο αυτό στον directory LAB06 του εργαστηρίου. Θα πρέπει να γράψετε ένα πρόγραμμα το οποίο διαβάζει τα δεδομένα από το αρχείο αυτό και αντιστοιχεί τη κάθε στήλη σε έναν κατάλληλα ονοματισμένο array. Θα πρέπει να κάνετε το ίδιο διαβάζοντας το file απευθείας από την τοποθεσία του στο web σύμφωνα με την μέθοδο που παρουσιάστηκε στη διάλεξη 9. Θα πρέπει να δείτε και πάλι την σύνταξη της συνάρτησης loadtxt δίνοντας την εντολή help(np.loadtxt) όπου υποτίθεται ότι έχετε κάνει import την numpy as np.
    Θα μπορούσατε να χρησιμοποιήσετε την συνάρτηση της numpy, np.set_printoptions(precision=n) για να τυπώσετε τους πίνακες με συγκεκριμένο αριθμό, n, δεκαδικών ψηφίων. Μπορείτε να το δοκιμάσετε στο πρόγραμμά σας για να δείτε τους πίνακες που γεμίσατε διαβάζοντας τα δεδομένα από το file.
    Θα μπορούσατε να διαβάσετε συγκεκριμένες μόνο στήλες του αρχείου χρησιμοποιώντας την συνάρτηση loadtxt ως εξής:
    a,b = np.loadtxt("mydata.txt", usecols=(1,2), unpack=True,skiprows=5). Εδώ αγνοούμε τις 5 πρώτες γραμμές του file (skiprows=5). To πεδίο unpack χρησιμοποιείται για να δηλώσει ότι τα δεδομένα του αρχείου θα μεταφερθούν σε πίνακες ή κάτι ανάλογο.

  3. Να γράψετε ένα πρόγραμμα το οποίο αποθηκεύει τους πίνακες που διαβάσατε στην προηγούμενη άσκηση σε ένα file το οποίο ονομάζεται mydatawritten.txt. Θα πρέπει να γράψετε τα δεδομένα ως float αριθμούς με μήκος 12 ψηφίων με 1 ψηφίο μετά την υποδιαστολή. Μπορείτε να γράψετε όλους τους πίνακες δίνοντάς τους σε μία list και χρησιμοποιώντας την συνάρτηση zip. Με την χρήση της συνάρτησης zip μπορούμε να πακετάρουμε τους πίνακες των δεδομένων ως ένα πίνακα. Η σύνταξη για να το κάνουμε αυτό είναι list(zip(array1, array2, array3, array4)) και η εντολή που θα δίναμε  επομένως θα είχε τη μορφή: np.savetxt("mydatawritten.txt", list(zip(array1,array2, array3, array4))). H συνάρτηση zip επιστρέφει συνδιάζει τους πίνακες και επιστρέφει μία λίστα από tuples, με το i-th tuple να περιέχει το i-th στοιχείο του καθένα από τους πίνακες. Η συνάρτηση list πριν το zip χρειάζεται για να κατασκευάσουμε τη λίστα όλων των tuples.
    Μπορούμε να γράψουμε σχόλια στο file στο οποίο αποθηκεύουμε τους πίνακες. Τα σχόλια μπορούμε να τα γράψουμε σαν γραμματοσειρά (string) μεταβλητή και στην εντολή savetxt να χρησιμοποιήσουμε το πεδίο header=string_var. Να τροποποιήσετε το πρόγραμμά σας ώστε να επιτρέπει την γραφή των ακόλουθων σχολίων ακριβώς όπως εμφανίζονται παρακάτω:

  6. O διονυμικός παράγοντας (nk)\binom{n}{k}    είναι ένας ακέραιος που ισούται με: (nk)=n!k!(n−k)!\binom{n}{k}=\frac{n!}{k!(n-k)!} όπου (n0)=1\binom{n}{0}=1 όταν k=0.
     (α) Χρησιμοποιώντας τον τύπο του διονυμικοπυ παράγοντα να γράψετε μια συνάρτηση Python, binomial(n,k) που υπολογίζει τους διονυμικούς παράγοντες και δεδομένα n και k. Προσέξτε ώστε η συνάρτησή σας επιστρέφει αποτέλεσμα που είναι ακέραιος και την σωστή τιμή 1 όταν k=0
    (β) Χρησιμοποιήστε τη συνάρτησή σας για να γράψετε ένα πρόγραμμα το οποίο τυπώνει τις πρώτες 20 γραμμές του τριγώνου Pascal. Η n-ιοστή γραμμή του τριγώνουν Pascal περιέχει n+1 αριθμούς οι οποίοι είναι οι διονυμικοί παράγοντες (n0),(n1),..,(nn)\binom{n}{0}, \binom{n}{1} και \binom{n}{n}
