  1. Μία μπάλα ρίχνεται κατακόρυφα προς τα πάνω  από ύψος h0 από την επιφάνεια του εδάφους και με αρχική ταχύτητα υ0. Όπως ξέρουμε, η θέση του h, και η ταχύτητά του σε μια οποιαδήποτε χρονική στιγμή δίνονται από τις εξισώσεις h = h0 + υ0t - gt2/2  και  υ = υ0 - gt, όπου g = 9.8m/s2. Να γράψετε ένα πρόγραμμα σε python το οποίο να υπολογίζει το ύψος h και την ταχύτητα υ σε μια χρονική στιγμή t. Θεωρήστε ως δεδομένα ότι η αρχική θέση του σώματος είναι h0 =1.6m και η αρχική του ταχύτητα είναι υ0 = 14.2m/s. Το πρόγραμμά σας θα πρέπει να τυπώνει τις τιμές του ύψους και της ταχύτητας μετά από 0.5sec κίνησης. Διαφοροποιήστε το πρόγραμμά σας ώστε να βρίσκει τη θέση και ταχύτητα μετά από 2.0sec κίνησης.   2. Μια δευτεροβάθμια εξίσωση έχει γενική μορφή αx2 + bx + c = 0, η οποία έχει δύο λύσεις x = [-b± sqrt(b2 - 4ac)]/2a. (α) Δίνοντας τα a, b και c, να γράψετε ένα πρόγραμμα σε python το οποίο δίνει τις δύο λύσεις της δευτεροβάθμιας. Να εκφράσετε τις σταθερές a, b και c ως μεταβλητές κινητής υποδιαστολής (float) και δείξτε ότι το πρόγραμμά σας δίνει τις σωστές λύσεις για ορισμένες περιπτώσεις που οι λύσεις είναι πραγματικοί αριθμοί, (η διακρίνουσα είναι θετική ή μηδέν). Χρησιμοποιήστε την συνάρτηση print για να τυπώσετε τις δύο λύσεις.
    (β) Γράφοντας το πρόγραμμα με αυτόν τον τρόπο (χρήση float), το πρόγραμμά σας θα δώσει σφάλμα όταν η διακρίνουσα είναι αρνητική και οι λύσεις μιγαδικές. Για παράδειγμα, εξετάστε τι θα σας δώσει το πρόγραμμά σας όταν α = 1, b = 2 και το c = 3. Μπορείτε να διορθώσετε το πρόβλημα αυτό χρησιμοποιώντας στο πρόγραμμά σας εντολές ανάθεσης της μορφής: a = a + 0j αφού έχετε πρώτα αναθέσει στο α κάποια τιμή float. Με τον τρόπο αυτό το πρόγραμμά σας μπορεί να τρέξει για οποιαδήποτε τιμή των σταθερών a, b και c. Χρησιμοποιήστε και πάλι την συνάρτηση print για να δείτε τις τιμές των λύσεων.     [συγκρίνετε με το lab02_prob02.py]
