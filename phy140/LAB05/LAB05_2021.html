  1. Δεδομένης μιας μεταβλητής string, να γράψετε τις εντολές που:
    (α) Τυπώνει τον πρώτο χαρακτήρα της γραμματοσειρά
    (β) Τυπώνει τον τελευταίο χαρακτήρα της γραμματοσειράς
    (γ) Τυπώνει τον τελευταίο χαρακτήρα με την χρήση της μεθόδου len()
    (δ) Βρίσκει σε ποια θέση της γραμματοσειράς υπάρχει το πρώτο κενό
    (ε) Τυπώνει μόνο την πρώτη λέξη της μεταβλητής string. Οι λέξεις χωρίζονται από κενά. Να μην χρησιμοποιήσετε την συνάρτηση split().     [Προτεινόμενη λύση]

  2. Δεδομένης μιας σταθεράς string που δίνει ο χρήστης από το πληκτρολόγιο

    (β) Γράψτε την εντολή για να εκτυπώσετε τους 4 τελευταίους χαρακτήρες της [Προτεινόμενη λύση]

  4. Έστω ότι ένας χρήστης σας δίνει μια μεταβλητή string που περιέχει το όνομα και το επίθετό του π.χ. name_str="Fotios Ptochos". Ποιές εντολές θα χρησιμοποιούσατε για να εξάξετε το όνομά του και το επίθετό του σε δύο διαφορετικές μεταβλητές τύπου string;  Ποιές εντολές θα χρησιμοποιούσατε για να φτιάξετε μια νέα μεταβλητή με το επίθετο και το όνομα δοσμένα με αυτή την σειρά; [Προτεινόμενη λύση]

  5. Γράψτε ένα πρόγραμμα σε Python που θα ζητά από τον χρήστη την τιμή μιας μεταβλητής string και δύο χαρακτήρες c1 και c2. To πρόγραμμα θα αναζητά στην μεταβλητή string κάθε παρουσία του χαρακτήρα c1 και θα τον αντικαθιστά με τον χαρακτήρα c2. Για παράδειγμα αν ο χρήστης δώσει  astring="Albert Einstein is a scientist" και χαρακτήρες c1='i' και c2='_' τότε το πρόγραμμά σας θα δημιουργήσει μια μεταβλητή string newstring='Albert E_nste_n _s a sc_ent_st'.  Θα πρέπει να χρησιμοποιήσετε ένα while loop. [Προτεινόμενη λύση]

  6. Δημιουργήστε μια λίστα που περιέχει 20 τυχαίους ακεραίους αριθμούς μεταξύ 0 και 1000. Βρείτε το άθροισμα,  μέσο όρο, τυπική απόκλιση των στοιχείων της λίστας χρησιμοποιώντας συνδυασμό εντολών Python και Numpy. [Προτεινόμενη λύση]

  7. Έστω Α ένας ΜxΝ πίνακας όπου Μ και Ν δίνονται από το πληκτρολόγιο. Γράψτε ένα πρόγραμμα το οποίο δέχεται τον Α και επιστρέφει τον ανάστροφό του Β. (Ανάστροφος ενός πίνακα Α είναι ο πίνακας Β=ΑΤ, τα στοιχεία του οποίου είναι Βij = Aji για κάθε i,j). Θεωρήστε ότι ο πίνακας Α έχει στοιχεία τυχαίους ακεραίους στο διάστημα [20,500]. Προσπαθήστε να κάνετε το ίδιο χρησιμοποιώντας τη δομή list comprehension. Σημείωστε ότι μπορείτε να μετατρέψετε την αρχική list σε numpy array και κατόπιν να χρησιμοποιήσετε την μέθοδο ArrayName.T ή ArrayName.transpose() για να πάρετε τον ανάστροφο (όπου Τ είναι για το transpose). Χρησιμοποιείστε σαν παράδειγμα μια αρχική λίστα Α 3x4 με στοιχεία  [[4,18,23,1],[54,32,2,8],[9,11,21,13]]. Θα πρέπει να γράψετε το πρόγραμμά σας με τέτοιο τρόπο ώστε τα στοιχεία της αρχικής λίστας να δίνονται από το πληκτρολόγιο με τη βοήθεια ενός file το οποίο εισάγει τα δεδομένα στο πρόγραμμά σαs. [Προτεινόμενη λύση]
