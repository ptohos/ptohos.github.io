ΕΞΑΣΚΗΣΗ

Τα ακόλουθα προγράμματα να σταλούν με e-mail στο τέλος του εργαστηρίου με subject: LAB07. To tgz αρχείο θα πρέπει να περιέχει τις ασκήσεις του εργαστηρίου και να έχει το όνομα username_lab07.tgz όπου username το όνομα χρήστη που έχετε στο πανεπιστήμιο.
 

  1. Συνάρτηση 1: Θα πρέπει να γράψετε ένα πρόγραμμα το οποίο υπολογίζει τη συνάρτηση sinc(x) = sin(x)/x για τιμές του x στο διάστημα  [-5., 5.] με βήμα 0.1. Η συνάρτησή σας θα πρέπει να δέχεται ως όρισμα ένα προς ένα τα στοιχεία  του array x που δημιουργείτε. Θα διαπιστώσετε γρήγορα ότι η συνάρτηση θα σας επιστρέψει σφάλμα και στην πραγματικότητα ΝΑΝ (Not A Number) όταν προσπαθήσετε να υπολογίσετε την τιμή της συνάρτησης για x=0. Δηλαδή η συνάρτηση μοιάζει να μήν ορίζεται για x=0. Ωστόσο, αν χρησιμοποιήσετε το ανάπτυγμα Taylor  του ημιτόνου ως προς x=0 τότε sin(x) = x - x3/3! + x5/5! +... και η συνάρτηση μπορεί να γραφεί με την μορφή sinc(x) = 1 - x2/3! + x4/5! +... . Με βάση την παρατήρηση αυτή, μπορείτε να τροποποιήσετε το πρόγραμμά σας και να γράψετε την συνάρτηση sinc(x) ώστε να μην παρουσιάζεται το πρόβλημα για x=0.

  2. Συνάρτηση 2: Στο προηγούμενο πρόβλημα εισαγάγατε στη συνάρτηση ξεχωριστά κάθε στοιχείο του array των τιμών του x. Ωστόσο θα ήταν χρήσιμο να περάσετε ολόκληρο τον array σαν ορισμα της συνάρτησης και να σας επιστρέφει έναν πίνακα με τις τιμές της sinc(x) για τα αντίστοιχα στοιχεία του x. Πρέπει να προσέξετε ότι οι εντολές ελέγχου δεν μπορούν να λειτουργήσουν με ολοκληρους πίνακες αλλά μόνο με συγκεκριμένες τιμές. 

  3.  Να γράψετε ένα πρόγραμμα το οποίο δέχεται μια λίστα 5 float type αριθμούς από το πληκτρολόγιο, τους εισάγει σε κάποιoν numpy array τον οποίο θα πρέπει να αποθηκεύσετε σε ένα file με το όνομα lab07_prob03.dat και θα πρέπει να έχει 3 ψηφία μετά την υποδιαστολή. Οι αριθμοί μπορεί να είναι [12.4, 21.3, 48.4, 2.2, 0.5].

  4. Να γράψετε το περιεχόμενο ενός file test.txt to οποίο είναι μια στήλη με τη γραμματοσειρά data1, data2, data3, data4, data5, data6, data7 σε μια στήλη, σε ένα άλλο file newtest.txt όπου δεν τυπώνεται την 5η γραμμή του αρχικού file, δηλαδή θα πρέπει να είναι: data1,data2, data3, data4, data6, data7 και πάλι στη μορφή μιας στήλης

  5. Να γράψετε ένα πρόγραμμα το οποίο ελέγχει αν ένα σημείο πέφτει στην περιοχή ενός ορθογωνίου που ορίζεται εσείς. Το πρόγραμμά σας θα πρέπει να διαβάζει τις συντεταγμένες του σημείου και να ελέγχει αν πέφτει μέσα στο ορθογώνιο και πέφτει να τυπώνει τις συντεταγμένες και το ένα μύνημα ότι πέφτει μέσα στην περιοχή του ορθογωνίου χρησιμοποιώντας κατάλληλο format.
     
  6. Χρησιμοποιήστε το αρχείο orbit.csv για να διαβάσετε τα δεδομένα σε 4 numpy arrays με ονόματα time, posx, posy και posz. Υπολογίστε τη μέση τιμή των τιμών του κάθε πίνακα, και την τυπική απόκλιση που ορίζεται ως 1Ν∑(xvaluei−xmean)2\sqrt{\frac{1}{Ν}∑\left(xvalue_{i}-x_{mean}\right)^2} όπου το πλήθος των τιμών του κάθε array, και xmean η μέση τιμή των τιμών αυτών. Θα πρέπει να κάνετε τους υπολογισμούς της μέσης τιμής και τυπικής απόκλισης σε κάποιες συναρτήσεις κατάλληλα ορισμένες. Αποθηκεύστε τους πίνακες σε ένα νέο file storedata.dat με τις τιμές να διαχωρίζονται με κενά.
