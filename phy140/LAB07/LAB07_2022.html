    import matplotlib.pyplot as plt
    import numpy as np
    names = ['groupA', 'groupB', 'groupC']

    plt.figure()
    plt.subplot(131)     # 1 γραμμή - 3 στήλες - 1ο γράφημα

    plt.subplot(132)    #  1 γραμμή - 3 στήλες - 2ο γράφημα

    plt.subplot(133)    # 1 γραμμή - 3 στήλες - 3ο γράφημα

    plt.suptitle("Many graphs")

  7. Γραμμική παρεμβολή (linear interpolation): Χρησιμοποιείται για καλύτερη εύρεση αριθμητικών αποτελεσμάτων. Αυτό γιατί στους περισσότερους αριθμητικούς υπολογισμούς, τα επιθυμητά αποτελέσματα δεν προκύπτουν σαν συνεχείς συναρτήσεις αλλά σα ένα πεπερασμένο σύνολο διακριτών σημείων. Για παράδειγμα, υποθέστε ότι θέλετε να υπολογίσετε αριθμητικά τη τροχιά ενός σώματος  το οποίο κινείται κάτω από την επίδραση ενός ομοιόμορφου πεδίου βαρύτητας. Η αναλυτική λύση βέβαια σας είναι πολύ γνωστή:

    x = x0 + v0x t
    y = y0 + v0yt - 1/2 gt2

    όπου x0 και y0 είναι οι αρχικές συντεταγμένες (τη χρονική στιγμή t = 0), ενώ v0x και v0y είναι οι συνιστώσες της ταχύτητας του σώματος τη χρονική στιγμή t = 0 και g είναι η σταθερή επιτάχυνση εξαιτίας της βαρύτητας. Ωστόσο η αριθμητική λύση επειδή συνήθως υπολογίζεται από μια ομάδα διακριτών χρονικών στιγμών ti η κατάσταση του συστήματος περιγράφεται από τις τριάδες (ti, xi, yi).  Χωρίς κάποια επιπλέον φυσική πληροφορία δεν ξέρουμε τίποτα σχετικά με τη λύση της εξίσωσης κίνησης σε κάποια μεταγενέστερη χρονική στιγμή.

    Πολλές φορές χρειάζεται να εξαγάγουμε κάποια πληροφορία  σχετικά με την κατάσταση (λύση) του συστήματος σε μεταγενέστερη χρονική στιγμή. Για παράδειγμα για να υπολογίσουμε το διάστημα που κάλυψε ένα σώμα x  όταν  y = y0 (για παράδειγμα όταν το σώμα έπεσε στο έδαφος) κάποια χρονική στιγμή tx η οποία ωστόσο δεν αντιστοιχεί σε κάποια από τα προκαθορισμένα διαστήματα ti + dt. Επομένως θα πρέπει να βρούμε ένα τρόπο για να υπολογίσουμε τη χρονική αυτή στιγμή.  

    Μαθηματικά, είναι πάντοτε δυνατό να προσαρμόσουμε μια συνεχή καμπύλη μέσω όλων των Ν σημείων που έχουμε (γενικά Ν σημεία καθορίζουν ένα μοναδικό πολυώνυμο Ν-1 βαθμού). Η καμπύλη αυτή μπορεί να χρησιμοποιηθεί για να ορίσουμε τις ενδιάμεσες τιμές της αριθμητικής λύσης. Ωστόσο χρειάζεται προσοχή στο να θυμάστε ότι η μαθηματική αυτή καμπύλη αποτελεί μια προσαρμογή ή διαφορετικά η καλύτερη πρόβλεψη για την αληθινή λύση βασιζόμενη στην υπόθεση ότι είναι συνεχής.

    Η απλούστερη προσέγγιση μέσω προσαρμογής που θα χρησιμοποιήσουμε απλά συνδέει διαδοχικά σημεία με ευθύγραμμα τμήματα.  Η προσέγγιση αυτή ονομάζεται γραμμική παρεμβολή (linear interpolation). H εξίσωση της ευθείας μεταξύ δύο διαδοχικών σημείων με συντεταγμένες (ti, yi) και (ti+1, yi+1) είναι:  

    y(t) = yi + (t-ti) [ (yi+1 - yi) / (ti+1 - ti)]      (εξ. 1)

    H γραμμική αυτή συνάρτηση x(t) αποτελεί την προσέγγισή μας για την αληθινή λύση. Από τις εξισώσεις κίνησης έχουμε τις θέσεις yi και yi+1 για τις χρονικές στιγμές ti και ti+1. Έστω ότι μεταξύ των δύο αυτών διαδοχικών στιγμών το βλήμα πέρασε από κάποια γνωστή θέση y=yo (π.χ. έπεσε στο έδαφος , οπότε y=0 και τότε η θέση yi είναι μεγαλύτερη του 0 ενώ η θέση yi+1 < 0). Χρησιμοποιώντας την εξ. 1, μπορούμε να λύσουμε ως προς t για να βρούμε τη χρονική στιγμή που y=yo.

    Θα έχουμε: t(yo) = ti + (yo - yi) [ (ti+1 - ti) / (yi+1 - yi) ]

    Αντικαθιστώντας το y0 παίρνουμε τις παρεμβαλλόμενη τιμή του t. Αυτό θα μπορούσαμε να το κάνουμε και για ζευγάρια τιμών x και y και να εφαρμόζαμε μια ανάλογη σχέση με την εξ. 1 αντικαθιστώντας όπου t το x.

    Από τις τιμές που διαβάσατε από το αρχείο, υπολογίστε την κλίση της ευθείας που θα περνά από τα σημεία όπως φαίνεται στο log-log γράφημα.

    Όπως συζητήσαμε στη διάλεξη, η ευθεία αυτή είναι της μορφής: log(y) = log(A) + λ*log(x).     (εξ. 2)

    Για να βρείτε την κλίση θα πρέπει να υπολογίσετε τους λογαρίθμους των συντεταγμένων δύο σημείων (x1,y1) και (x2,y2) και να εφαρμόσετε τη σχέση που δίνει την κλίση της ευθείας: λ = [log(y1)-log(y2)] / [log(x1)-log(x2)].

    Με βάση την κλίση μπορούμε να χρησιμοποιήσουμε τις συντεταγμένες (x3,y3) ενός τρίτου σημείου για να βρούμε την τιμή της τεταγμένης log(A). Θα έχουμε: log(A) = log(y3) - λ*log(x3).

Μπορείτε να μετατρέψετε το log(y) σε y χρησιμοποιώντας τη σχέση y = exp[log(y)].

    Κάντε το γράφημα του y ως προς x που θα πρέπει να είναι η ευθεία που περνά από τα σημεία.  Κατασκευάστε ακόμα την ευθεία που είναι κάθετη στον x-άξονα στο σημείο x=1.
