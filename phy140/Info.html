Οδηγίες για απομακρυσμένη σύνδεση με τους υπολογιστές του εργαστηρίoυ

 

Μπορείτε να συνδεθείτε με τους υπολογιστές του εργαστηρίου ενώ βρίσκεστε εκτός Πανεπιστημίου Θα πρέπει χρησιμοποιήσετε το λογισμικό VPN και την εφαρμογή Microsoft Remote Connection.

  1. Κατεβάστε το λογισμικό VPN από την ιστοσελίδα της βιβλιοθήκες του Πανεπιστημίου. Επιλέξτε το λογισμικό κατάλληλο για το λειτουργικό σύστημα του προσωπικού σας υπολογιστή.
    ***Σε περίπτωση που δεν μπορείτε να συνδεθείτε στο portal του Πανεπιστημίου και ο προσωπικός σας Η/Υ δουλεύει με microsoft windows, το pdf αρχείο με τις πληροφορίες της εγκατάστασης μπορείτε να το κατεβάσετε σε αυτό το link (windows). Θα πρέπει να κατεβάσετε επίσης το executable της εφαρμογής FortiClient από αυτό το link (windows). Αν έχετε υπολογιστή Apple (macbook air, macbook pro) που τρέχει macosx version 10.5 ή μεγαλύτερο, θα βρείτε τις οδηγίες εγκατάστασης σε αυτό το link και το αντίστοιχο dmg file. Αν έχετε υπολογιστή που τρέχει Linux τότε ακολουθήστε αυτό το link. Ακολουθήστε τα επόμενα βήματα όπως περιγράφονται παρακάτω.
  2. Μετά την εγκατάσταση ξεκινήτε την εφαρμογή FortiClient κάνοντας διπλό click στο εικονίδιο του shortcut που έχει δημιουργηθεί στο Desktop σας.
  3. Στο παράθυρο διαλόγου που ανοίγει, εισάξετε το username σας στο Πανεπιστήμιο και το password του e-mail σας στο Πανεπιστήμιο
  4. Αν το λειτουργικό του προσωπικού σας υπολογιστή είναι Windows, τότε στο Windows search box πληκτρολογήστε το "Remote Desktop" και κάντε διπλό click στην εφαρμογή για να ξεκινήσει η εφαρμογή.
  5. Στο παράθυρο του διαλόγου που εμφανίζεται θα πρέπει να εισάξετε ως όνομα ή IP διεύθυνση του υπολογιστή που θέλετε να συνδεθείτε, την IP διεύθυνση ενός από τους υπολογιστές του εργαστηρίου. Οι διευθύνσεις αυτές έχουν τα ακόλουθα νούμερα: 194.42.7.202 (έως 216) και 194.42.7.239 (έως 249).
  6. Θα πρέπει να εισάξετε επίσης το username σας στους Η/Υ του εργαστηρίου (σε όλους σας θα πρέπει να είναι ίδιο με αυτό του email στο πανεπιστήμιο) και κατόπιν τον κωδικό πρόσβασης που δεν είναι ίδιος με αυτό του e-mail στο πανεπιστήμιο. Είναι αυτός που σας δόθηκε στο Τμήμα.
  7. Όταν συνδεθείτε θα ανοίξει το Desktop του CENTOS, οπότε μπορείτε να βρείτε την εφαρμογή terminal και να την ανοίξετε. Τα υπόλοιπα είναι όπως όταν χρησιμοποιείτε τους υπολογιστές στο εργαστήριο.
  8. Για να στείλετε τις ασκήσεις που κάνετε σε e-mail, μπορείτε να κάνετε tar τις ασκήσεις και κατόπιν να ανοίξετε το chrome και να επισκεφθείτε την ιστοσελίδα του e-mail του Πανεπιστημίου και να στείλετε το e-mail.
  9. Για όσους έχουν υπολογιστές με λειτουργικό IOS ή mac os x, θα χρειαστείτε να κατεβάσετε την εφαρμογή του Microsoft Remote Desktop από το AppleStore. 

 


Οδηγίες για εγκατάσταση του λειτουργικού CENTOS7 στον προσωπικό σας υπολογιστή

Οι παρακάτω οδηγίες μπορούν να χρησιμοποιηθούν για να εγκαταστήσετε το λειτουργικό CENTOS7 στους προσωπικούς σας υπολογιστές. Θα χρειαστείτε τα ακόλουθα βήματα:
  1. Θα πρέπει να κατεβάσετε και να εγκαταστείσετε αρχικά το λογισμικό VirtualBox της Oracle.

  2. Κατεβάστε το CENTOS7 από την ιστοσελίδα του Τμήματος (Θα πρέπει να έχετε ξεκινήσει το FortiClient VPN για να κατεβάσετε το file).

  3. Κάντε διπλό click στο εικονίδιο του shortcut της εφαρμογής VirtualBox. Στο παράθυρο που ανοίγει, επιλέξτε NEW, τη δημιουργία μια νέας μηχανής.



  4. Στον διάλογο που ακολουθεί, δώστε αρχικά το όνομα της μηχανής που θέλετε (π.χ. centos) και επιλέξτε ότι είναι λειτουργικό Linux RedHat 64 bit και πιέστε Next



    Αν δεν έχετε αυτή την επιλογή και η μόνη που σας δίνει είναι 32 bits τότε θα πρέπει να κάνετε κάποια άλλα βήματα (δείτε παρακάτω) και να επιστρέψετε στο βήμα αυτό για να ολοκληρώσετε την δημιουργία της μηχανής.

  5. Στο επόμενο βήμα θα πρέπει να επιλέξετε τη μνήμη που θα χρησιμοποιήσετε. Μετατοπίστε τoν δείκτη έως ότου δείτε 2048Mb RAM. Πατήστε κατόπιν Next



  6. To επόμενο στάδιο θα πρέπει να εισάξετε το λειτουργικό της μηχανής. Επιλέξτε την τελευταία επιλογή "Use an existing virtual hard disk file". Πιέζοντας το κίτρινο φάκελο δεξιά από το κουτί επιλογής, θα πρέπει να βρείτε το file centos7v.vdi που κατεβάσατε στο βήμα 2 και να το επιλέξετε για εισαγωγή. Πατήστε Next για να ολοκληρωθεί η δημιουργία της μηχανής.



  7. Μετά τη δημιουργία της μηχανής, κάντε διπλό click στο όνομα της μηχανής που εμφανίζεται  στην αριστερά πλευρά του menu του VirtualBox ή επιλέξτε το όνομα της μηχανής και πατήστε το Start.

  8. Οταν ξεκινήσει η μηχανή θα πάει σε μια σειρά από διαδικασίες εκκίνησης και αν δεν υπάρχει κάποιο πρόβλημα τότε θα πρέπει να σας δώσει την login οθόνη του CENTOS με username student. O κωδικός πρόσβασης είναι: ##Phy140$$. Την πρώτη φορά που θα κάνετε login θα πρέπει να αλλάξετε τον κωδικό αυτό σε κάτι που μπορεί να θυμάστε εύκολα. Ενώ έχετε κάνει login, επιλέξτε από το Applications, το System Tools και στο submenu επιλέξτε terminal. Στο terminal window, πληκτρολογήστε την εντολή passwd για να αλλάξετε τον κωδικό πρόσβασης. Αρχικά θα πρέπει να εισαγάγετε τον παλιό κωδικό και κατόπιν τον καινούργιο τον οποίο θα ζητήσει να επιβεβαιώσετε. Προσέξτε ότι όταν πληκτρολογείτε τον κωδικό δεν εμφανίζεται οτιδήποτε στην οθόνη.

  9. Υπάρχει ακόμα ένας account που έχει δημιουργηθεί. Ο account αυτός έχει όνομα root και είναι ο λογαριασμός του administrator. Μπορείτε να κάνετε login με τον ίδιο τρόπο χρησμιποιώντας το username root και ως password το  ##Phy140$$, το οποίο θα πρέπει να αλλάξετε με την ίδια διαδικασία που περιγράφηκε στο βήμα 8.

  10. Μπορείτε να δημιουργήσετε τον δικό σας account με username ίσως ίδιο με αυτόν που έχετε στο πανεπιστήμιο για να έχετε παντού το ίδιο username και να μην μπερδεύεστε. Για να το κάνετε αυτό μπορείτε να κάνετε login ως root (βήμα 9). Αφού κάνετε login, ανοίξτε ένα terminal window και δώστε την εντολή adduser username όπου username αυτό που θέλετε να έχετε (για παράδειγμα θα έδινα adduser phy140). Κατόπιν δίνετε την εντολή passwd username (όπου username αυτό που επιλέξατε στην προηγούμενη εντολή - phy140 για το παράδειγμα).

  11. Στο βήμα αυτό θα έρθετε σε περίπτωση που δεν μπορέσατε να κάνετε την εγκατάσταση είτε γιατί δεν βρήκατε Redhat 64bit ή κατά την εκκίνηση της μηχανής κάτι δεν πήγε καλά. Θα πρέπει να κάνετε κάποια αλλαγή στα BIOS του υπολογιστή σας για να επιτρέψει την δημιουργία της νέας μηχανής.
    Για το τι ακριβώς πρέπει να κάνετε ακολουθήστε τα βήματα από τις δύο πρώτες μεθόδους που περιγράφονται στο ακόλουθο link. Κάτι ανάλογο περιγράφεται επίσης και σε αυτό το link. Μετά την ενεργοποίηση του virtualization στα bios θα πρέπει να ακολουθήσετε τα βήματα 3-9 για την δημιουργία της μηχανής.

 

Οδηγίες για εγκατάσταση επιπλέον python modules


1) Επειδή το version του centos είναι σχετικά παλιό, θα ήταν καλό να κάνετε update τα λογισμικά πακέτα που είναι τοποθετημένα. Θα πρέπει να δώσετε την εντολή su  ακολουθούμενη από το password του root. Κατόπιν δώστε την εντολή:
yum update -y

2) Θα χρειαστεί να εγκαταστήσετε μερικά ακόμα python modules που είναι ιδιαίτερα χρήσιμα. Θα πρέπει να δώσετε την εντολή:
python3 -m pip install --user numpy matplotlib scipy ipython jupyter pandas sympy nose


3) Για animated graphs θα χρειαστεί να εγκαταστήσετε επίσης το ffmpeg-python πακέτο: python3 -m pip install --user ffmpeg-python

Θα πρέπει επίσης να εγκαταστήσετε το ffmpeg πακέτο. Για να το κάνετε αυτό ακολουθήστε τα επόμενα βήματα:

(α) Στο terminal δώστε την εντολή
       su    και θα πρέπει να δώσετε κατόπιν το root password ποου βλέπετε παραπάνω όταν κάνατε εγκατάσταση του centos

(β) Δώστε κατόπιν την εντολή:

      yum localinstall --nogpgcheck https://download1.rpmfusion.org/free/el/rpmfusion-free-release-7.noarch.rpm -y

(γ) Τέλος δώστε την εντολή:

       yum install ffmpeg ffmpeg-devel -y

 

4) Για μηχανική μάθηση (machine learning) θα χρειαστείτε το πακέτo scikit-learn το οποίο μπορείτε να εγκαταστήσετε ως:

     python3 -m pip install --user -U scikit-learn

 

 

 

Οδηγίες για εγκατάσταση python3

Πριν κάνετε εγκατάσταση της Python3 σε υπολογιστή mac θα πρέπει να εγκαταστήσετε το πακέτο Xcode και το command Line Tools (θα πρέπει να έχετε Apple account - τα links που δίνονται είναι για version 14). Άν έχετε ήδη εγκαταστήσει το λογισμικό πακέτο Xcode θα χρειαστεί να εγκαταστήσετε το command Line Tools δίνοντας σε terminal την εντολή xcode-select  --install


Για να εγκαταστήσετε εύκολα διάφορα πακέτα του λειτουργικού Linux θα πρέπει να εγκαταστήσετε το λογισμικό Homebrew. Για να εγκαταστήσετε Homebrew, ανοίξτε ένα terminal και πλητρολογήστε την ακόλουθη εντολή:


/bin/bash -c "$(curl -fsSL http://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

Θα δείτε τις αλλαγές που πρόκειται να συμβούν και θα σας ρωτήσει αν θέλετε να προχωρήσει στις προτεινόμενες αλλαγές. Όταν ολοκληρωθεί η εγκατάσταση του Homebrew, θα πρέπει να εισάξετε τον φάκελο της εφαρμογής Homebrew στο λίστα των φακέλων που ψάχνει για διάφορες εφαρμογές. Αυτό σημαίνει ότι θα πρέπει να τροποποιήσετε την μεταβλητή PATH. Αυτό γίνεται εύκολα αν βάλετε την επόμενη εντολή στο file με όνομα .profile που βρίσκεται στο homedirectory σας:


export PATH="/usr/local/opt/python/libexec/bin:$PATH


Αφού εκτελέσετε το προηγούμενο βήμα είστε έτοιμοι να προχωρήσετε με την εγκατάσταση της Python 3. Αυτό το κάνετε με την εντολή:


brew install python


Μετά την εγκατάσταση, δώστε απλά στο terminal την εντολή python3. Για να δείτε ότι όντως έχετε την version 3 της Python δώστε την εντολή python --version


 


  Οδηγίες για τη χρήση της εντολής tar

 

Η εντολή tar επιτρέπει τη αποθήκευση πολλών αρχείων σε ένα και μόνο νέο αρχείο ενώ παράλληλα ελαττώνει το χώρο που απαιτείται για φύλαξη διαφόρων αρχείων. Δύο από τους πιο χρήσιμες εφαρμογές της εντολής είναι η δημιουργία αρχείου και η επαναφορά αρχείων από ένα τέτοιο tar αρχείο στην κανονική τους μορφή.


Δημιουργία tar file:  

Πληκτρολογήστε την εντολή:   tar  -czvf   <filename>.tgz   file1.py  file2.py   file3.py

 

Θα μπορούσατε αντί να χρησιμοποιήσετε το όνομα του κάθε αρχείου ξεχωριστά να αντικαταστήσετε τα ονόματα όλων των αρχείων που έχουν την ίδια προέκταση (π.χ. .py) με τον wild χαρακτήρα (*). Για παράδειγμα η πιο πάνω εντολή μπορεί να γραφεί:

 tar -czvf  <filename>.tgz *.py  *.dat  *.txt

Όλα τα αρχεία με προέκταση .py, *.dat και *.txt    (file1, file2, file3) που βρίσκονται στο φάκελο που βρίσκεστε θα αρχειοθετηθούν στο tar αρχείο.

Θα μπορούσατε να κάνετε tar ένα ολόκληρο directory καθώς και τους  subdirectories που υπάρχουν μέσα σε αυτόν δίνοντας την εντολή tar -czvf <filename>.tgz  <directory name>/

 

Εξαγωγή αρχείων από tgz file:

Για να εξάγετε τα files από ένα tgz file δώστε την εντολή: tar -xzvf <flename>.tgz Αυτή η εντολή θα εξάγει όλα τα files ή directories που βρίσκονται μέσα στο file και θα αποθηκεύσει στο directory που βρίσκεστε. Θα μπορούσατε να βγάλετε μόνο κάποια συγκεκριμένα files αν ξέρετε τα ονόματά τους. Για παράδειγμα tar -xzvf <filename>.tgz file1.py θα εξάγει το file file1.py που βρίσκεται στο tgz file.


Περιεχόμε ενός tgz file:

Για να δείτε το περιεχόμενο ενός tgz file δώστε την εντολή tar -tzvf <filename>.tgz 

 

Για περισσότερες πληροφορίες σχετικά με την εντολή tar και το τι σημαίνουν τα γράμματα (c, x, v, f) που χρησιμοποιήσαμε παραπάνω μπορείτε να πληκτρολογήσετε την εντολή man tar&nbs (man είναι η αντίστοιχη εντολή help σε περιβάλλον Linux/Unix)

Οδηγίες για την μεταφορά αρχείων μεταξύ υπολογιστών

Για την μεταφορά αρχείων μεταξύ υπολογιστών του προσωπικού σας laptop και των υπολογιστών του εργαστηρίου του Τμήματος (IP διεύθυνση: 194.42.7.202 (έως 216) και 194.42.7.239 (έως 249)). Θα πρέπει ωστόσο πριν το επιχειρήσετε να έχετε συνδεθεί με το VPN του Πανεπιστημίου. Για να κατεβάσετε το application του VPN του πανεπιστημίου επισκεφθείτε την ιστοσελίδα του Πανεπιστημίου και ακολουθήστε τις οδηγίες ανάλογα με το λειτουργικό του προσωπικού σας υπολογιστή

  • Ανοίξτε ένα παράθυρο terminal και πληκτρολογήστε την εντολή:
    scp  filename (που θέλετε να μεταφέρετε)   username@194.42.7.210:~/.

  • Η εντολή αυτή θα πάρει το filename που πληκτρολογήσατε και θα το μεταφέρει στο home directory σας στους υπολογιστές του εργαστηρίου του Τμήματος. Προσέξτε ότι μετά την ηλεκτρονική διεύθυνση υπάρχει (:) και στο μετά το (/) υπάρχει (.) που σημαίνει ότι το file θα έχει το ίδιο όνομα. username είναι το όνομα χρήστη στους υπολογιστές του Τμήματος

  • Για να μεταφέρετε ένα ολόκληρο directory και subdirectory tree κάντε το ακόλουθο:
     scp -r dir  username@194.42.7.210:~/.
    Για παράδειγμα:  scp  -r Lab07  fotis@194.42.7.210:~/.  θα μεταφέρει όλα τα files του directory Lab07 συμπεριλαμβανομένων και τυχόν υπο-directories σε ένα άλλο μηχάνημα και θα δημιουργήσει τον ίδιο directory (Lab07) καθώς και όλους τους υπό-directories. dir είναι το όνομα του φακέλου που θέλετε να μεταφέρετε και username το όνομα του χρήστη. 

  • Σημειώστε ότι όταν πατήσετε <enter> θα σας ρωτήσει αν θέλετε να σώσετε  την ηλεκτρονική διεύθυνση που θέσατε. Απαντήστε yes. Κατόπιν θα σας ρωτήσει το  password που αντιστοιχεί στο username που θέσατε. Πληκτρολογήστε το password που έχετε. Καθώς πληκτρολογείτε το password, δεν θα εμφανίζεται στην οθόνη.

Η εντολή scp είναι η πλέον ενδεδειγμένη γιατί κάνει ασφαλή αντιγραφή (Secure CoPy scp). Δηλαδή το password δεν μπορεί να "κλαπεί" κατά την πληκτρολόγηση.



Οδηγίες για την εκτύπωση αρχείων στον εκτυπωτή του εργαστηρίου


Για να εκτυπώσετε αρχεία στον εκτυπωτή του εργαστηρίου θα πρέπει να κάνετε τα ακόλουθα:

1) Θα χρειαστεί να ανοίξετε ένα terminal


2) Δώστε τις ακόλουθες εντολές:


    mkdir .config/autostart
    cp /usr/local/pc-client-linux.sh.desktop .config/autostart/
    cp /usr/local/pc-client.properties ~/.


H διαδικασία αυτή χρειάζεται να γίνει μόνο μία φορά από οποιονδήποτε υπολογιστή του εργαστηρίου και όχι κάθε φορά που θέλετε να εκτυπώσετε κάποιο αρχείο.
