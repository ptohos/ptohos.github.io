  1. Ανοίξτε ένα παράθυρο terminal και πληκτρολογήστε την εντολή python3 για να μπείτε σε περιβάλλον της PYTHON. Δημιουργήστε μία λίστα με το όνομα mynumbers και τοποθετείστε στη λίστα τα στοιχεία 2*i-1 όπου το i πέρνει τιμές από 0 έως και 4. Χρησιμοποιείστε τη συνάρτηση plot της matplotlib.pyplot για να κάνετε το γράφημα της λίστας αυτής. Προσέξτε ότι αν δεν δώσετε μία λίστα για τιμές του x-άξονα, η PYTHON θεωρεί ότι η λίστα που εισάγεται στην εντολή plot αφορά τιμές του y-άξονα και οι τιμές του x-άξονα δημιουργούνται αυτόματα θεωρώντας ότι είναι οι θέσεις των στοιχείων της λίστας που χρησιμοποιείται στον y-άξονα. To γράφημά σας θα πρέπει να έχει επομένως τιμές στον x-άξονα που είναι 0, 1, 2, 3 και 4. Ονοματίστε κατάλληλα τον x- και y- άξονα και δώστε στο γράφημα τον τίτλο My first plot
    Μπορείτε να κάνετε το ίδιο γράφημα, αλλά αντί της συμπαγούς γραμμής να χρησιμοποιήσετε κύκλους που είναι χρώματος κόκκινο. 

  2. Tα γραφήματα στο προηγούμενο παράδειγμα μπορούν να μπουν σε ένα παράθυρο (figure) το οποίο χωρίζεται σε γραμμές και στήλες (σαν να είναι πίνακας) και σε κάθε κελί του πίνακα βάζουμε ένα συγκεκριμένο γράφημα. Θα μπορούσαμε να κάνουμε τα plots είτε το ένα δίπλα στο άλλο (αυτό σημαίνει μία γραμμή με δύο στήλες) ή το ένα κάτω από το άλλο (2 γραμμές με 1 στήλη).  Όταν κάνουμε το γράφημα μπορούμε να δώσουμε την εντολή plt.subplot(IrIcIp) όπου Ιr είναι ο αριθμός γραμμής,  Ic o αριθμός της στήλης και Ιp o αριθμός του plot.
    Δοκιμάστε για παράδειγμα τις ακόλουθες εντολές:
    import matplotlib.pyplot as plt
    import numpy as np
    names = ['groupA', 'groupB', 'groupC']
    values = [1,10,100]
    plt.figure()
    plt.subplot(131)     # 1 γραμμή - 3 στήλες - 1ο γράφημα
    plt.plot(names,values,'og')
    plt.subplot(132)    #  1 γραμμή - 3 στήλες - 2ο γράφημα
    plt.plot(names,values,'r--')
    plt.subplot(133)    # 1 γραμμή - 3 στήλες - 3ο γράφημα
    plt.plot(names,values,'^-')
    plt.suptitle("Many graphs")
    plt.show()

    Αν κάποιος ήθελα να σώσει το παραπάνω figure σε ένα file θα μπορούσε να δώσει την εντολή plt.savefig("myFirstPlot.pdf")

  3. Δοκιμάστε να γράψετε ένα πρόγραμμα σε PYTHON το οποίο χρησιμοποιεί μία συνάρτηση με όνομα myfun(t) που δέχεται σαν input όρισμα έναν numpy array και επιστρέφει έναν numpy array με τιμές που δίνονται από την εξίσωση e-t*cos(2*π*t). H συνάρτηση αυτή θα πρέπει να καλείται από το κύριο πρόγραμμα κατά την κατασκευή κάποιων γραφημάτων.
    Συγκεκριμένα θα πρέπει να κάνετε το γράφημα της myfunc(x1) συναρτήσει του array x1 που έχει τιμές στο εύρος 0.0 έως 5.0 με βήμα 0.1. Θα πρέπει να κάνει επίσης το γράφημα της myfunc(x2) συναρτήσει των τιμών του array x2 με τιμές στο διάστημα 0.0 έως 5.0 με βήμμα 0.02. 
    Τα δύο γραφήματα θα πρέπει να φαίνονται σανδύο καμπύλες στο ίδιο plot. Η πρώτη καμπύλη θα πρέπει να έχει σημείο με κύκλους χρώματος μπλέ, ενώ η δεύτερη καμπύλη θα πρέπει να είναι μια μαύρη συμπαγής γραμμή. 
    Το plot αυτό θα πρέπει να είναι στην 1η θέση ενός παράθυρου γραφημάτων που θα έχει 2 γραμμές και 1 στήλη. Στην 2η θέση του παράθυρου αυτού θα πρέπει να βάλετε το γράφημα της cos(2*π*x2) ως προς τον array x2, με καμπύλη διακεκομμένη χρώματος κόκκινου.
