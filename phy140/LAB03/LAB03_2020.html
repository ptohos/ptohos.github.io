ΕΞΑΣΚΗΣΗ

  • Δημιουργήστε ένα sub-directory με το όνομα LAB03 στον οποίο και να γράψετε τα προγράμματα με τα οποία θα ασχοληθείτε στο σημερινό εργαστήριο.  

  • Στο τέλος του εργαστηρίου θα πρέπει να κάνετε tar τα προγράμματα σας και να τα στείλετε με e-mail στο fotis@ucy.ac.cy. Όλα τα προγράμματα θα πρέπει να σταλούν σε ένα e-mail με subject  LAB03. Το attached file θα πρέπει να έχει όνομα με τη μορφή  <username>_lab03.tgz
    Η εντολή  tar έχει την ακόλουθη σύνταξη:     tar -czvf   <filename>.tgz   filename1  filename2 filename3 κλπ.  Στην προκειμένη περίπτωση <filename>.tgz θα είναι username_phy140_lab03.tgz και filename1(2,3) το όνομα του file της αντίστοιχης άσκησης. Αν τα αρχεία σας έχουν όλα το ίδιο ακρονύμιο (π.χ. askisi1.py, askisi2.py, askisi3.py κλπ) μπορείτε να δώσετε την εντολή με την μορφή tar -czvf <filename>tgz *.py Στην περίπτωση αυτή όλα τα αρχεία με extension .py θα γραφούν στο tgz αρχείο που φτιάχνετε.

  • Για να ανοίξετε ένα tgz αρχείο και να βγάλετε τα αρχεία που υπάρχουν μέσα σε κάποιο φάκελο δώστε την εντολή tar -xzvf <filename>.tgz . Η εντολή θα κάνει extract (x) όλα τα αρχεία στο directory που βρίσκεστε.

  • Μην ξεχάσετε να γράψετε σε κάθε file το ονοματεπώνυμό σας.

  • Όσες ασκήσεις δεν προλάβετε να τελειώσετε κατά τη διάρκεια του εργαστηρίου μπορείτε να τις επιστρέψετε αργότερα αλλά πριν το επόμενο εργαστήριο.

Για παράδειγμα, αν δώσετε τον αριθμό 10, τότε το άθροισμα θα πρέπει να είναι 55. Υπόδειξη: χρησιμοποιώντας τη συνάρτηση int() μπορείτε να μετατρέψετε μια σταθερά string σε integer. Ανάλογα μπορείτε να χρησιμοποιήσετε τη συνάρτηση float() για μετατροπή σταθεράς σε τύπο float.
