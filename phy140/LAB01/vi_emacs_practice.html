Εξάσκηση με τους editors vi και emacs

  1. Αντιγράψτε το αρχείο firstdoc.txt στον subdirectory LAB01 (για να το κάνετε αυτό τοποθετήστε το mousse στο link και πιέστε το δεξί πλήκτρο του mousse και επιλέξτε Download link file as. Ακολουθείστε τις ερωτήσεις για να το αντιγράψετε στο σωστό directory).

  2. Επεξεργαστείτε το αντίγραφο του αρχείου που έχετε με τον editor vi.

  3. Πηγαίνετε στο τέλος του κειμένου και πληκτρολογείστε την ακόλουθη παράγραφο:

My cell phone rings. Should I answer it? No I will continue reading and typing. Better send a msg
My mothr wants something. I don't want to talk now. Oh look how nice everybody is typing
Nah I'm tuping too. Let's have a rest. I'm already exhausted. I can't type like this all day. I better go
do my differential equation homework. This looks cool. I skeep typing and typing. Let me try
something something else now.

  1. Διορθώστε τα 4 ορθογραφικά λάθη (1 σε κάθε γραμμή,  msg->message, mothr->mother, tuping->typing, skeep->keep)
    της παραγράφου που προσθέσατε. Προσθέστε την λέξη out στην 2η γραμμή της 1ης παραγράφου και σβήστε το επιπλέον JUNGLE στη τελευταία γραμμή της 1ης παραγράφου και το επιπλέον something στην τελευταία γραμμή της 2ης παραγράφου.

  2. Προσθέστε τη φράση "About time!"   στο τέλος της 2ης παραγράφου. 

  3. Σβήστε την πρόταση "Nah I'm typing too." (3η γραμμή της 2ης παραγράφου) και ξαναφτιάξτε την παράγραφο.

  4. Αντικαταστήστε όλες τις λέξεις "my" με την λέξη "our".

  5. Εναλλάξτε τη θέση των δυο παραγράφων.

  6. Αποθηκεύστε το file με όλες τις αλλαγές και βγείτε από το vi.

  7. Επαναλάβετε την ίδια διαδικασία χρησιμοποιώντας τον editor emacs. Ποιος είναι ευκολότερος;

  8. Μπορείτε να γράψετε ένα απλό πρόγραμμα Fortran (το HELLO WORLD πρόγραμμα) να το κάνετε compile και να το τρέξετε όλα ενώ βρίσκεστε στον emacs?
    [    Για να κάνετε compile ένα πρόγραμμα Fortran πληκτρολογείτε g77 -o <name>.x <name>.f όπου name είναι ο όνομα του file που περιέχει το πρόγραμμά σας]
    [Για να τρέξετε ένα πρόγραμμα χρειάζεται να πληκτρολογήσετε την εντολή ./<name>.x (προσοχή την τελεία πριν το σύμβολο /)]
    [Το πρόγραμμα hello.f θα μπορούσε να είναι το ακόλουθο:
                  program hello
                  print *, "Hello world"
                  end
    ]

  9. Αν θέλετε ένα περισσότερο εκτενές μάθημα για τον vi δοκιμάστε να τρέξετε "vimtutor". Για ένα εκτενές μάθημα στον emacs, πληκτρολογείστε M-x help-with-tutorial ενώ βρίσκεστε σε περιβάλλον emacs.
